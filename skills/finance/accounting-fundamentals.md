# Accounting Fundamentals

This skill teaches and applies core accounting principles — from the accounting equation to financial statement analysis — whenever a user needs help understanding or managing business finances.

## Core Accounting Workflow

### 1. Identify the Context
Determine what the user needs:
- **Conceptual understanding** (e.g., what is a debit?)
- **Transaction recording** (e.g., how do I record a sale?)
- **Statement analysis** (e.g., what does this balance sheet tell me?)
- **Reconciliation or error-finding** (e.g., my books don't balance)

### 2. Anchor to the Accounting Equation
Always ground explanations in the foundational equation:

**Assets = Liabilities + Equity**

Every transaction must keep this equation balanced. Use this as the starting point for any journal entry or account analysis.

### 3. Apply Debit/Credit Rules
Use this reference for every transaction:

| Account Type | Debit Effect | Credit Effect |
|---|---|---|
| Asset | Increase ↑ | Decrease ↓ |
| Liability | Decrease ↓ | Increase ↑ |
| Equity | Decrease ↓ | Increase ↑ |
| Revenue | Decrease ↓ | Increase ↑ |
| Expense | Increase ↑ | Decrease ↓ |

- Every transaction has at least one debit and one credit
- Total debits must always equal total credits

### 4. Record Journal Entries Correctly
Structure every entry:
1. Date of transaction
2. Account(s) debited — listed first, at full left margin
3. Account(s) credited — listed second, indented
4. Brief narrative description (memo)

Example:
```
Date: 2024-01-15
  DR  Accounts Receivable    $5,000
      CR  Revenue                     $5,000
  Memo: Invoiced client for consulting services
```

### 5. Map Entries to Financial Statements
Show how transactions flow:
- **Income Statement**: Revenue and Expense accounts
- **Balance Sheet**: Asset, Liability, and Equity accounts
- **Cash Flow Statement**: Actual cash movement from operations, investing, financing

### 6. Explain Key Concepts on Demand
Cover these topics with concrete examples when asked:
- **Accrual vs. Cash basis**: When to recognize revenue and expenses
- **Matching principle**: Expenses recognized in the same period as related revenue
- **Depreciation**: Spreading asset cost over useful life (straight-line, declining balance)
- **Accounts Payable/Receivable**: Money owed to/from the business
- **Trial balance**: Verifying debits equal credits before closing the books
- **Chart of accounts**: Numbered account structure (1000s=Assets, 2000s=Liabilities, etc.)

### 7. Diagnose Imbalances or Errors
If books don't balance:
1. Verify debits = credits in every journal entry
2. Check for transposition errors (e.g., $450 entered as $540)
3. Confirm all entries posted to the correct accounts
4. Re-run trial balance after each correction

## Output Format

**For concept explanations:**
- Plain-language definition (1–2 sentences)
- Concrete real-world example
- How it connects to the accounting equation or financial statements

**For journal entries:**
- Formatted DR/CR table with account names and amounts
- Memo line
- Confirmation that debits equal credits
- One sentence explaining which financial statement is affected
