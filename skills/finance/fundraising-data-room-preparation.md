# Fundraising Data Room Preparation

This skill audits, organizes, and structures all financial and legal documents required for investor due diligence into a standardized data room format ready for Series A through growth-stage fundraises.

## Data Room Audit & Organization Workflow

### Step 1: Gather Current Document Inventory
Ask the founder to list or upload all documents currently available. Categorize each into one of these core folders:

- **Corporate & Legal**: Certificate of Incorporation, bylaws, board minutes, stockholder agreements, NDAs
- **Cap Table & Equity**: Cap table (fully diluted), option pool schedule, SAFEs/convertible notes outstanding, 409A valuation
- **Financials**: Last 3 years P&L, balance sheets, cash flow statements, monthly actuals for current year
- **Projections**: 3–5 year financial model, key assumptions document, scenario analysis (base/bull/bear)
- **Business**: Pitch deck, executive summary, product roadmap, competitive landscape
- **Customers & Revenue**: Customer list (anonymized if needed), MRR/ARR breakdown, churn data, top 10 customer contracts
- **Team**: Org chart, founder bios, key employee offer letters, advisor agreements
- **IP & Technology**: Patent filings, trademark registrations, key vendor/SaaS contracts
- **Tax & Compliance**: Last 3 years tax returns, any outstanding litigation or liabilities

### Step 2: Identify Gaps
Cross-reference the provided inventory against the required document list above. Flag every missing document with:
- **MISSING**: Document does not exist yet
- **INCOMPLETE**: Exists but needs updating (e.g., cap table not fully diluted, projections older than 6 months)
- **SENSITIVE**: Exists but needs redaction or gating before sharing

### Step 3: Prioritize by Investor Stage
Rank gap severity based on fundraise stage:
- **Seed/Pre-Seed**: Prioritize cap table, pitch deck, financial model, incorporation docs
- **Series A**: All of the above plus 2+ years actuals, customer contracts, 409A, full option schedule
- **Series B+**: Everything above plus audited financials, board minutes, detailed unit economics, data room access logs

### Step 4: Build the Remediation Plan
For each gap, specify:
1. What needs to be created or updated
2. Who is responsible (founder, CFO, legal counsel, accountant)
3. Estimated time to complete
4. Whether it blocks investor access or can be added post-NDA

### Step 5: Structure the Data Room
Recommend a folder hierarchy using this naming convention:
```
[CompanyName]_DataRoom_[Month][Year]/
├── 01_Corporate_Legal/
├── 02_CapTable_Equity/
├── 03_Financial_Statements/
├── 04_Financial_Projections/
├── 05_Business_Overview/
├── 06_Customers_Revenue/
├── 07_Team/
├── 08_IP_Technology/
└── 09_Tax_Compliance/
```

Recommend a data room platform based on context (Docsend for early-stage, Datasite or Intralinks for later-stage, Google Drive as a fallback).

### Step 6: Access Control Recommendations
- Tier 1 (initial outreach): Pitch deck + executive summary only
- Tier 2 (post-NDA or serious interest): Full financials, cap table, customer data
- Tier 3 (final diligence): Tax returns, board minutes, employee agreements
Flag any documents that should never be shared without legal counsel review.
