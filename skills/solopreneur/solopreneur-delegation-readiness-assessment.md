# Solopreneur Delegation Readiness Assessment

This skill evaluates a solopreneur's recurring tasks against documentation completeness and strategic value criteria to identify what can be safely delegated today, what needs prep work first, and what must stay with the owner.

## Assessment Workflow

### Step 1: Collect the Task Inventory
Ask the user to list their recurring tasks (weekly, monthly, or per-project). If they haven't listed them, prompt with: "Walk me through a typical week — what tasks do you do repeatedly that aren't core strategy or client work?"

### Step 2: Score Each Task on Two Axes

**Axis A — Strategic Value (1–5)**
- 5: Requires your unique judgment, relationships, or vision (e.g., offer creation, pricing decisions, client strategy calls)
- 3: Involves your preferences but could be templated (e.g., content editing, proposal formatting)
- 1: Purely mechanical, no judgment required (e.g., scheduling, invoicing, transcription)

**Axis B — Documentation Readiness (1–5)**
- 5: Step-by-step SOP exists, examples included, edge cases covered
- 3: Informal notes or a rough process exists but gaps remain
- 1: Lives entirely in your head — no written process, no examples

### Step 3: Apply the Delegation Matrix

| Strategic Value | Documentation Readiness | Verdict |
|---|---|---|
| 1–2 | 4–5 | ✅ Delegate Now |
| 1–2 | 2–3 | 🔧 Document First, Then Delegate |
| 1–2 | 1 | ⚠️ High Priority to Document |
| 3 | 4–5 | 🤝 Delegate with Close Review |
| 3 | 1–3 | 🔧 Build SOP Before Delegating |
| 4–5 | Any | 🔒 Keep — Owner Only |

### Step 4: Identify Documentation Gaps
For every task scored below 4 on documentation readiness, identify the specific missing elements:
- Missing trigger: What starts this task?
- Missing steps: Where does the process get fuzzy?
- Missing examples: Are there templates, samples, or past outputs?
- Missing exceptions: What edge cases must the delegate know?

### Step 5: Recommend Delegation Method
For each delegatable task, suggest the appropriate handoff path:
- **AI tool**: Repetitive, text-based, or data tasks (e.g., transcription, first-draft emails, formatting)
- **Async VA**: Tasks requiring human judgment but no real-time interaction (e.g., inbox triage, scheduling, research)
- **Specialist contractor**: Tasks requiring domain expertise (e.g., bookkeeping, graphic design, video editing)

## Output Format

Produce a structured report with the following sections:

**1. Task Delegation Summary Table**
List every task with its Strategic Value score, Documentation Readiness score, and Verdict label from the matrix.

**2. Delegate Now (Quick Wins)**
Bullet list of tasks ready for immediate handoff. For each: task name, recommended delegation method (AI/VA/contractor), and one sentence on why it's ready.

**3. Document First (Next 2 Weeks)**
Bullet list of high-delegation-potential tasks blocked only by missing SOPs. For each: task name, specific documentation gap to fill, and estimated time to document (be realistic: most SOPs take 30–90 minutes).

**4. Keep for Now (Owner-Only)**
Brief list of tasks to retain with one-line reason for each.

**5. Immediate Next Action**
Single, specific action the solopreneur should take in the next 24 hours — name
