# Content-to-Inquiry Attribution Log

Captures inbound inquiry data tied to specific content pieces or platforms and surfaces patterns so the solo operator knows exactly what to create more of.

## Attribution Logging Workflow

1. **Capture the inquiry event**
   - Ask: What did the prospect say they saw, read, or heard before reaching out?
   - Record the exact content piece (post title, video topic, newsletter subject, podcast episode) if known, or the platform/topic if the specific piece is unknown.
   - Note the date of inquiry and the date the content was published if available.

2. **Classify the entry**
   - Platform: LinkedIn / Instagram / Newsletter / Podcast / YouTube / Referral / Other
   - Content type: Short-form post / Long-form article / Video / Email / Story / Thread
   - Topic/theme: Tag with 1–3 topic keywords (e.g., "pricing," "case study," "personal story")
   - Inquiry quality: Cold lead / Warm lead / Ready-to-buy (based on operator's assessment)

3. **Log the entry**
   - Add the entry to the running attribution log in the output format below.
   - If a content piece already has prior inquiries, increment its count rather than creating a duplicate row.

4. **Trigger pattern analysis when asked or when log reaches 5+ entries**
   - Identify which platform, content type, or topic has the highest inquiry count.
   - Flag any content piece that generated 2+ inquiries as a "proven converter."
   - Note which topics produced warm/ready-to-buy leads vs. cold leads.

5. **Produce a prioritized recommendation**
   - State the single highest-converting content type or topic.
   - Recommend one immediate next action (e.g., "Repurpose your pricing transparency post into a newsletter," or "Record a video on the same case study topic").

## Output Format
