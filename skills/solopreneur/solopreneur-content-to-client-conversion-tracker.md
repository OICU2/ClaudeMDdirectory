# Content-to-Client Conversion Tracker

Builds a practical, no-code-required tracking system for solopreneurs to identify exactly which content pieces, topics, and platforms are generating real client inquiries and revenue — so they can double down on what works and cut what doesn't.

## Tracking System Setup

### Step 1: Intake Audit
Ask the solopreneur for:
- All active content platforms (newsletter, LinkedIn, YouTube, podcast, blog, etc.)
- How clients currently reach out (DM, email, contact form, booking link)
- Whether they have any existing CRM, spreadsheet, or notes on past clients
- Approximate number of new clients per month

### Step 2: Design the Tracking Touchpoints
Create a simple inquiry intake ritual: every time a potential client reaches out, capture:
1. **Source** — Where did they find you? (Ask directly: "How did you come across my work?")
2. **Specific content** — Did they mention a post, episode, or article? Which one?
3. **Topic/theme** — What problem or topic drew them in?
4. **Platform** — Which channel delivered that content?
5. **Outcome** — Did inquiry convert to paid work? (Yes / No / Pending)
6. **Revenue** — If yes, how much?

### Step 3: Build the Master Tracker
Produce a ready-to-use spreadsheet schema with these exact columns:
| Date | Client Name | Platform | Content Piece | Topic/Theme | Inquiry Type | Converted? | Revenue | Notes |

Provide Google Sheets formula to auto-calculate:
- Conversion rate per platform: `=COUNTIFS(D:D,"[Platform]",G:G,"Yes")/COUNTIFS(D:D,"[Platform]")`
- Total revenue per platform: `=SUMIF(D:D,"[Platform]",H:H)`
- Top converting topics: use pivot table instructions

### Step 4: Tagging System
Define 3–5 topic tags based on the solopreneur's content pillars. Every content piece gets one primary tag. This reveals which *themes* convert — not just which posts went viral.

### Step 5: Monthly Review Ritual
Set a recurring 20-minute monthly review to answer:
- Which platform sent the most inquiries this month?
- Which content piece or topic triggered the most conversations?
- What is the revenue-per-platform this quarter?
- Which content got engagement but zero inquiries? (Cut or reframe it.)

### Step 6: Quick-Reference Feedback Loop Rules
- If a platform has 3+ months of zero inquiries → deprioritize or cut
- If one topic tag converts at 2x the rate of others → create more content on that theme
- If a specific content piece keeps getting mentioned → repurpose it across all platforms
- Always ask new clients "how did you find me?" as a non-negotiable intake step

## Output Format

Produce the following four deliverables in sequence:

1. **Tracker Schema** — The exact spreadsheet column headers, described in a copyable table, with 3 example filled-in rows using realistic solopreneur scenarios.

2. **Inquiry Script** — A 2–3 sentence natural-language question to ask every new inquiry to capture source data without sounding robotic. Provide one version for email and one for DM.

3. **Monthly Review Checklist** — A bulleted 5-question checklist the solopreneur runs each month, with the specific metrics they are looking for in each answer.

4. **90-Day Insight Report Template** — A simple fill-in-the-blank summary they can complete after 90 days of tracking, with slots for: top platform, top topic, revenue by source, and one content decision they will make based on the data.
