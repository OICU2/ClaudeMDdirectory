# Client Project Debrief Template Builder

Builds a comprehensive post-project debrief document that surfaces what worked, what failed, scope and budget variance, and specific process improvements to apply to future client engagements.

## Debrief Building Process

1. **Gather project context** — Ask for: project name, client name, project type, original scope/deliverables, agreed timeline, agreed budget or rate, actual end date, actual hours or cost, and team members involved.

2. **Pull performance data** — Calculate and record:
   - Timeline variance: planned end date vs. actual end date (days over/under)
   - Budget/hours variance: estimated vs. actual (percentage and raw delta)
   - Scope changes: additions, removals, or pivots that occurred mid-project

3. **Diagnose what worked** — Prompt for 3–5 specific wins: workflows, tools, communication patterns, or deliverables that landed well. Push for specifics, not generalities ("weekly Loom updates reduced revision rounds" not "communication was good").

4. **Diagnose what didn't work** — Prompt for 3–5 specific friction points: bottlenecks, misalignments, underestimates, client behavior patterns, or internal failures. For each, identify the root cause (scope ambiguity, underpricing, poor onboarding, etc.).

5. **Analyze scope variance** — For every scope addition, document: what was added, who requested it, whether it was billed, and whether a change order was issued. Flag any pattern of unbilled scope creep.

6. **Extract lessons learned** — Convert each friction point into a forward-looking process fix. Format: "Because [problem], next time we will [specific action]."

7. **Generate profitability snapshot** — Compute effective hourly rate if hours are provided (total revenue ÷ actual hours). Flag if below target rate. Note which phases consumed the most unplanned time.

8. **Draft client relationship notes** — Summarize communication style, decision-making speed, revision behavior, and a recommendation: pursue similar work, adjust terms next time, or decline renewal.
