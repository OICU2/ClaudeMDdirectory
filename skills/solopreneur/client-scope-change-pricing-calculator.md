# Scope Change Pricing Calculator

Helps freelancers and solopreneurs calculate fair additional fees when clients request work beyond the original project scope, producing a ready-to-send change order summary.

## Scope Change Pricing Workflow

**Step 1 — Gather project baseline**
Ask for the following if not already provided:
- Original project scope (brief description)
- Original agreed price and hourly or daily rate
- Current project completion percentage
- Contract terms relevant to scope changes (kill fee, change order clause, revision limits)

**Step 2 — Define the new request**
Clarify the change being requested:
- What specifically is the client asking for?
- Is this an addition, modification, or replacement of existing work?
- Has work on the original scope already been completed that conflicts with the change?

**Step 3 — Estimate time impact**
Calculate time required for the change:
- New hours to complete the added/changed work
- Rework hours if existing work must be revised or discarded
- Buffer: add 20% to raw estimate to account for coordination, revisions, and communication overhead
- Total billable hours = (new hours + rework hours) × 1.2

**Step 4 — Calculate the fee**
Apply this formula:
- **Base change fee** = Total billable hours × hourly rate
- **Disruption fee** (if change derails timeline or requires rescheduling): add 10–25% of base change fee
- **Rush premium** (if client needs the change completed urgently): add 25–50% of base change fee
- **Total scope change fee** = Base + applicable premiums

**Step 5 — Check against contract terms**
- If a change order clause exists, confirm the fee aligns with its provisions
- If no clause exists, note that this sets a precedent and recommend formalizing terms going forward
- Flag if the change is significant enough to warrant a contract amendment rather than a simple change order

**Step 6 — Assess relationship context**
Ask: Is this a long-term client or a one-time project? Offer a brief note on whether a goodwill discount (5–10%) is strategically appropriate, without recommending they undervalue their work.

## Output Format

Produce a structured **Scope Change Pricing Summary** with these sections:

**1. Change Request Summary** (2–3 sentences describing what the client is asking for)

**2. Time Breakdown**
- New work hours: X hrs
- Rework hours: X hrs
- Buffer (20%): X hrs
- **Total billable hours: X hrs**

**3. Fee Calculation**
| Item | Amount |
|---|---|
| Base change fee (X hrs × $Y/hr) | $Z |
| Disruption fee (if applicable) | $Z |
| Rush premium (if applicable) | $Z |
| **Total scope change fee** | **$Z** |

**4. Contract Alignment Note** (1–2 sentences on whether existing terms cover this and any flags)

**5. Ready-to-Send Change Order Message**
A professional, direct email/message the freelancer can copy and send to their client that:
- Acknowledges the request positively
- States the additional fee clearly
- Explains what it covers in plain language
- Requests written approval before work begins
- Is 150 words or fewer

**6. Recommendation** (1–2 sentences on whether to proceed, negotiate, or revisit contract terms)
