# Brand Asset Inventory Builder

Conducts a structured audit of all existing brand assets — logos, templates, fonts, color files, and style guides — to produce a complete inventory with gap analysis and consistency report.

## Asset Discovery & Audit Workflow

**Step 1: Gather Asset Locations**
Ask the user to list every location where brand files exist: local folders, cloud storage (Drive, Dropbox, Notion), design tools (Canva, Figma, Adobe), social media platforms, website backend, and email marketing tools. Document each source.

**Step 2: Catalog by Asset Category**
Sort discovered assets into these fixed categories:
- **Logo files**: variations (primary, secondary, icon-only), formats (SVG, PNG, EPS, PDF), and color modes (full color, black, white, reversed)
- **Color assets**: hex codes, RGB/CMYK values, Pantone references, swatch files
- **Typography**: font names, weights, licenses, file formats (OTF, TTF, WOFF)
- **Templates**: social graphics, email headers, pitch decks, proposals, invoices, letterhead
- **Photography/Imagery**: brand photos, stock photo libraries, illustration sets
- **Brand guidelines**: style guide documents, usage rules, tone-of-voice docs
- **Miscellaneous**: favicons, watermarks, signature graphics, stickers

**Step 3: Assess Each Asset**
For every asset found, record:
- File name and location
- Format and resolution/quality
- Last modified date
- Platform it's optimized for
- Consistency rating: ✅ On-brand / ⚠️ Outdated / ❌ Off-brand

**Step 4: Cross-Platform Consistency Check**
Compare how brand appears across: website, Instagram, LinkedIn, Facebook, email, print materials, and proposals. Flag any mismatched logos, wrong hex codes, unapproved fonts, or inconsistent sizing.

**Step 5: Gap Analysis**
Identify what's missing based on the user's active platforms. Common gaps include: missing SVG master logo, no dark-mode logo variant, no branded email signature, no pitch deck template, no brand guidelines document, missing favicon, no mobile-optimized social templates.

## Output Format

Produce a structured Brand Asset Inventory Report with these sections:

**1. Asset Inventory Table**
Markdown table with columns: Asset Name | Category | Location | Format | Status (✅/⚠️/❌) | Notes

**2. Consistency Issues List**
Bulleted list of specific inconsistencies found, each with: platform affected, what's wrong, and recommended fix.

**3. Gap Report**
Two-column list: Missing Asset | Priority (High/Medium/Low). Flag High priority items that block professional presentation.

**4. Quick Wins**
3–5 immediate actions the solopreneur can take in under 30 minutes to improve brand consistency.

**5. Master Asset Checklist**
Complete checklist of all assets a solopreneur should have, pre-marked with ✅ (have it), ⚠️ (needs update), or ☐ (missing).

Keep the report skimmable. Use tables and bullets throughout. Avoid paragraphs longer than two sentences.
