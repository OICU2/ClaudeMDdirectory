# Efficiency Analysis

This skill audits operational processes to surface waste, bottlenecks, and underutilized capacity, then produces a ranked action plan for doing more with less.

## Analysis Workflow

1. **Define scope** — Clarify what system, process, or team is being analyzed. Ask for current inputs (headcount, time, budget, tools) and expected vs. actual outputs if not provided.

2. **Map the current state** — Break the operation into discrete steps or functions. For each step, identify:
   - Who owns it
   - How long it takes
   - How often it runs
   - What the failure or rework rate is

3. **Apply efficiency lenses** — Evaluate each step against these criteria:
   - **Redundancy**: Is this duplicated elsewhere?
   - **Automation gap**: Could a tool or script replace manual effort?
   - **Handoff friction**: Are delays caused by waiting, approvals, or unclear ownership?
   - **Underutilization**: Are resources (people, tools, time windows) sitting idle?
   - **Scope creep**: Is this step doing more than its original purpose requires?

4. **Quantify waste** — Estimate time or cost lost per inefficiency per week or month. Use rough math if exact data is unavailable; flag assumptions clearly.

5. **Rank opportunities** — Score each opportunity by:
   - Impact (high / medium / low)
   - Effort to fix (low / medium / high)
   - Speed to value (days / weeks / months)

6. **Identify quick wins** — Separate any high-impact, low-effort changes that can be acted on immediately without process redesign.

## Output Format

Produce a structured report with the following sections:

**Efficiency Analysis Report**

- **Scope**: One sentence defining what was analyzed
- **Baseline Metrics**: Current inputs vs. outputs (use estimates if needed, label them)
- **Inefficiency Inventory**: Bulleted list of identified waste points, each with a category label (redundancy, automation gap, handoff friction, underutilization, scope creep) and estimated cost in time or money
- **Opportunity Ranking**: Table with columns — Opportunity | Impact | Effort | Time to Value
- **Quick Wins**: Numbered list of 2–5 actions that can start this week
- **Recommended Next Steps**: 3 prioritized actions with an owner suggestion and success metric for each

Keep the report scannable. Use tables and bullets throughout. Aim for under 600 words unless the operation is complex. Flag every assumption made during analysis.
