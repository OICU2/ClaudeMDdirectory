# Facilities Management Planning

This skill generates comprehensive facilities management plans that organize space utilization, maintenance schedules, vendor assignments, safety inspections, and lease obligations into a single actionable document.

## Planning Workflow

1. **Gather Facility Profile** — Identify facility type (office, warehouse, retail, mixed-use), total square footage, number of floors/zones, occupancy count, and lease vs. owned status.

2. **Map Space Utilization** — Break facility into named zones (e.g., open workspace, conference rooms, server room, common areas, storage). Assign current use, capacity, and any underutilized or overloaded areas. Note any planned reconfigurations.

3. **Build Cleaning Schedule** — Define cleaning frequency per zone (daily, weekly, monthly, quarterly). Specify tasks per cycle: trash removal, surface disinfection, floor care, restroom service, deep cleans. Assign responsible vendor or internal staff.

4. **Create Safety Inspection Calendar** — List required inspections: fire extinguisher checks, sprinkler tests, emergency lighting, HVAC filters, elevator certifications, AED inspections, and any jurisdiction-specific requirements. Assign due dates, responsible party, and compliance documentation method.

5. **Document Lease Obligations** — If leased, extract key obligations: rent escalation dates, renewal option windows, CAM reconciliation deadlines, permitted use clauses, tenant improvement allowances, and notice requirements for termination or renewal. Flag any upcoming deadlines within 12 months.

6. **Assign Maintenance Vendors** — For each system (HVAC, plumbing, electrical, pest control, landscaping, janitorial, security, elevators), record vendor name, contract expiration, SLA terms, emergency contact, and last service date.

7. **Set Preventive Maintenance Triggers** — Define PM tasks tied to equipment age, usage cycles, or seasonal needs. Include HVAC filter replacements, roof inspections, parking lot sealing, window washing, and generator testing with specific intervals.

8. **Identify Open Issues and Priorities** — List any deferred maintenance, active work orders, code violations, or capital improvement needs. Rank by urgency (critical/high/medium/low) and estimated cost.

## Output Format

Produce a structured facilities management plan with the following sections:

**Facility Overview** — Name, address, type, size, occupancy, ownership/lease status, primary facility contact.

**Space Utilization Map** — Table with columns: Zone Name | Square Footage | Current Use | Capacity | Status (optimal/underused/overcrowded) | Notes.

**Cleaning Schedule** — Table with columns: Zone | Task | Frequency | Responsible Party | Last Completed.

**Safety Inspection Calendar** — Table with columns: Inspection Type | Required Frequency | Next Due Date | Responsible Party | Compliance Record Location.

**Lease Obligations Summary** (if applicable) — Table with columns: Obligation | Description | Deadline/Trigger Date | Action Required | Status.

**Vendor Directory** — Table with columns: System/Service | Vendor Name | Contract Expiry | SLA | Emergency Contact | Last Service Date.

**Preventive Maintenance Schedule** — Table with columns: Equipment/System | PM Task | Interval | Next Due | Assigned Vendor.

**Open Issues Log** — Table with columns: Issue | Location | Priority | Estimated Cost | Target Resolution Date | Owner.

Keep each section concise. Use tables for all structured data. Flag any critical deadlines or safety items in bold. Total plan should be scannable and ready to hand to a facilities coordinator or property manager.
