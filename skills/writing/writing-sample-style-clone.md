# Writing Sample Style Clone

This skill extracts the stylistic DNA from any provided writing sample and generates new content that faithfully mirrors that exact voice, tone, and structure.

## Style Analysis & Replication Workflow

**Step 1 — Deconstruct the Sample**
Analyze the provided writing across these dimensions:
- **Sentence structure**: Average length, variation patterns, use of fragments, run-ons, or complex clauses
- **Rhythm**: Cadence patterns, how sentences accelerate or decelerate, use of lists or parallel structure
- **Tone**: Formal/casual, warm/clinical, authoritative/conversational, ironic/earnest
- **Vocabulary**: Latinate vs. Anglo-Saxon words, jargon density, uncommon word frequency, contractions
- **Voice markers**: Signature phrases, transitional habits, punctuation quirks (em-dashes, ellipses, semicolons), first/second/third person preference
- **Paragraph behavior**: Length, how ideas are introduced and closed, use of white space

**Step 2 — Build the Style Profile**
Summarize findings in a brief internal profile before generating output:
- Dominant sentence length (short/medium/long/mixed)
- 3–5 vocabulary-level descriptors (e.g., "plain," "elevated," "technical," "colloquial")
- Tone in one phrase (e.g., "dry and wry," "warm but authoritative")
- 2–3 structural quirks to replicate

**Step 3 — Generate the New Content**
Write the requested content using the style profile as a constraint, not a suggestion:
- Mirror sentence length variation exactly — if the sample alternates short-long-short, do the same
- Use words at the same register; avoid upgrading or downgrading vocabulary
- Reproduce punctuation habits deliberately
- Match paragraph density and white space behavior
- If the original uses rhetorical questions, asides, or direct address — carry them over

**Step 4 — Calibrate on Request**
If the user says the output doesn't feel right, ask one targeted question (e.g., "Should it feel more casual?" or "Was the original punchier?") then re-generate with that single axis adjusted.

## Output Format

Produce two blocks:

**Style Profile** (4–6 bullet points, unlabeled header "Style Profile"):
Concise breakdown of the extracted stylistic traits — sentence rhythm, tone, vocabulary level, and 1–2 notable quirks. Keep this under 80 words.

**Generated Content**:
The new content in the cloned style, delivered without framing or apology. Length matches the user's request or defaults to roughly the same length as the provided sample. No meta-commentary about the style inside the generated text itself.
