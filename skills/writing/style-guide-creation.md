# Style Guide Creation

This skill produces a structured, ready-to-use writing style guide tailored to a specific brand, team, or publication when someone needs to standardize written communication.

## Workflow

1. **Gather brand context** — Ask for: organization name, industry, target audience, primary content types (blog, docs, marketing, social), and any existing brand guidelines or examples of approved content.

2. **Define voice and tone** — Establish 3–5 core personality attributes (e.g., "authoritative but approachable"). Write a one-paragraph voice description and specify tone variations by context (support docs vs. social posts vs. sales copy).

3. **Set grammar and mechanics rules** — Document explicit decisions on:
   - Oxford comma (use or omit)
   - Capitalization (title case vs. sentence case for headings)
   - Numbers (spell out vs. numerals, thresholds)
   - Punctuation preferences (em dashes, ellipses, exclamation marks)
   - Active vs. passive voice policy

4. **Define formatting standards** — Specify heading hierarchy, list style (bulleted vs. numbered and when), paragraph length targets, bold/italic usage rules, and link text conventions.

5. **Build terminology glossary** — Create three lists:
   - **Preferred terms**: Words and phrases the brand uses (e.g., "customer" not "user")
   - **Banned terms**: Words to avoid and why
   - **Industry-specific terms**: Definitions for jargon used internally

6. **Add inclusive language guidelines** — Cover gender-neutral language, accessibility considerations, and any cultural sensitivity rules relevant to the audience.

7. **Include examples** — For each major rule, provide a ✅ correct example and a ❌ incorrect example using realistic, brand-relevant sentences.

## Output Format

Produce a single Markdown document with this structure:

```
# [Brand Name] Writing Style Guide

## 1. Overview & Purpose
## 2. Our Voice & Tone
   - Core personality traits
   - Tone by context table
## 3. Grammar & Mechanics
   - Subsections per rule category
## 4. Formatting Standards
   - Headings, lists, length, emphasis
## 5. Terminology Glossary
   - Preferred | Avoid | Definitions
## 6. Inclusive Language
## 7. Quick Reference Cheat Sheet
   - One-page summary of key rules
```

Length: 600–1,500 words depending on brand complexity. Every rule must include at least one concrete example. The Quick Reference section must fit on a single printed page and summarize the 10 most critical rules.
