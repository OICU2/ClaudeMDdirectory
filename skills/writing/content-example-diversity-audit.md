# Content Example Diversity Audit

This skill scans a piece of writing for imbalance in examples, case studies, anecdotes, and citations, then delivers a structured report with specific gaps and replacement suggestions.

## Audit Workflow

1. **Extract all examples**: Identify every named company, person, study, statistic, anecdote, or case reference in the content. List them explicitly before analysis.

2. **Categorize each example** across these dimensions:
   - **Industry/sector** (tech, finance, healthcare, retail, etc.)
   - **Company size** (startup, SMB, enterprise, nonprofit, public sector)
   - **Geographic region** (North America, Europe, Global South, etc.)
   - **Demographic representation** (gender, age group, ethnicity where inferrable or stated)
   - **Source type** (academic study, journalist report, corporate blog, anecdote, personal experience, government data)
   - **Time period** (recent vs. dated examples)

3. **Flag concentration problems**: Mark any dimension where 60% or more of examples cluster in a single category. Flag any dimension with zero representation where diversity would be meaningful.

4. **Score overall diversity**: Rate each dimension as Balanced / Narrow / Missing.

5. **Generate replacement suggestions**: For each flagged gap, propose 2–3 specific alternative example types or directions (not fabricated examples — suggest what to look for or where).

6. **Check for pattern bias**: Note if examples consistently position certain groups as problems to solve vs. innovators, or if certain industries only appear in negative contexts.

## Output Format

Produce a structured audit report with these sections:

**Example Inventory**
A bulleted list of every example found, with its category tags in brackets.

**Diversity Scorecard**
A table with rows for each dimension (Industry, Company Size, Geography, Demographics, Source Type, Recency) and columns: Dominant Category | Coverage Gap | Score (Balanced/Narrow/Missing).

**Flagged Issues**
Numbered list of specific problems, each written as: *[Dimension]: [What's overrepresented] dominates — [what's missing] has no representation.*

**Recommendations**
For each flagged issue, 2–3 concrete directions for finding or framing alternative examples. Include the type of source to consult (e.g., "Look for a healthcare or education sector case study from a mid-size organization outside the US").

**Quick Summary**
2–3 sentences summarizing the most urgent gaps and the overall diversity health of the content.
