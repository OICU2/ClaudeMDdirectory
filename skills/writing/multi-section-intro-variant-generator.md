# Multi-Section Intro Variant Generator

Produces 4–6 structurally distinct opening paragraphs for a single article or report so editors can evaluate competing hooks before investing in full drafts.

## Variant Generation Workflow

1. **Extract the core brief**: Identify the article's topic, target audience, publication tone, and intended argument or thesis from the user's input. Ask one clarifying question if the topic is ambiguous; otherwise proceed.

2. **Select distinct hook archetypes**: Assign each variant a different opening strategy. Choose from:
   - **Anecdote/Scene**: Opens mid-action with a specific, sensory moment.
   - **Provocative statistic**: Leads with a striking number or data point.
   - **Counterintuitive claim**: Opens by challenging a common assumption.
   - **Direct address**: Speaks directly to the reader's problem or question.
   - **Historical/contextual frame**: Anchors the topic in time or precedent.
   - **Quotation-driven**: Opens with a sharp, relevant quote followed by immediate tension.

3. **Write each variant in full**: Each intro must be complete and polished—not a sketch. Match the publication's tone (formal, conversational, journalistic, academic) throughout.

4. **Maintain consistent thesis trajectory**: Every variant, regardless of hook, must logically lead toward the same core argument or topic. Hooks differ; destination does not.

5. **Label and annotate each variant**: Prefix each with its archetype name and a one-line rationale explaining why this hook could work for this piece.

## Output Format

Produce the following structure for each variant (repeat 4–6 times):

---

**Variant [N] — [Archetype Name]**
*Why this works: [One sentence explaining the strategic logic of this hook for this specific piece.]*

[Full opening paragraph, 75–150 words, publication-ready prose.]

---

After all variants, include a short section titled **Editor's Comparison Notes** that lists:
- Which variant is strongest for high-stakes/formal contexts
- Which variant is strongest for broad/general audiences
- One specific editing tip per variant (e.g., "Swap the statistic for a more recent source if available")

Total output length: 600–1,000 words depending on paragraph depth requested.
