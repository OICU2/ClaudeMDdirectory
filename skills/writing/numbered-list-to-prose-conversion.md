# Numbered List to Prose Conversion

This skill converts structured list content into smooth, readable narrative prose, preserving all original information, order, and logic while eliminating bullet points and numbered formatting.

## Conversion Workflow

1. **Audit the source material** — Read every list item and identify the logical relationships between them: sequential steps, parallel points, cause-and-effect chains, or grouped categories.

2. **Map groupings** — Cluster related items into natural paragraph units. Each paragraph should carry a coherent sub-theme derived from the original list structure.

3. **Write topic sentences** — Open each paragraph with a sentence that introduces the group of ideas it will cover, drawn directly from the list content rather than invented.

4. **Integrate list items as sentences** — Rewrite each bullet or numbered item as a full sentence. Use transitional language (e.g., "additionally," "as a result," "building on this," "finally") to connect items that were previously separated by bullets.

5. **Preserve all original information** — Do not omit, summarize away, or alter the meaning of any item. Numbers, names, specifics, and technical details must appear verbatim or with only grammatical adaptation.

6. **Eliminate list artifacts** — Remove all bullet characters, numbering, dashes, and indentation. Replace list parallelism with varied sentence structures where appropriate.

7. **Read for flow** — Do a final pass to ensure transitions between sentences and paragraphs feel natural and the text reads as intentional writing, not a disguised list.

## Output Format

- **Structure**: One or more prose paragraphs — the number of paragraphs should match the natural groupings in the source list
- **Length**: Approximately equivalent word count to the original list; no padding or compression unless explicitly requested
- **Style**: Neutral, clear expository prose unless the user specifies a tone (formal, conversational, academic, etc.)
- **No lists**: The output must contain zero bullet points, dashes, or numbered items
- **No commentary**: Deliver only the converted prose — no preamble like "Here is your converted text" unless asked
