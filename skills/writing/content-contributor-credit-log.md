# Content Contributor Credit Log

This skill builds a structured, publication-ready log of all contributors, editors, and quoted sources for a piece of content so no one is misattributed, overlooked, or improperly credited.

## Contributor Capture Workflow

1. **Identify the content scope** — Ask for the article, essay, report, or piece title and its primary author(s) if not provided.

2. **Collect contributor categories** — Prompt the user to supply names and roles across these groups:
   - **Primary authors/writers** — name, role, byline preference
   - **Editors** — name, editing level (copy, developmental, fact-check)
   - **Researchers/reporters** — name, contribution summary (e.g., "conducted interviews," "compiled data")
   - **Quoted sources** — name, title/affiliation, nature of contribution (interview, written statement, study)
   - **Photographers/illustrators/designers** — name, asset description
   - **Translators or transcribers** — name, language or task

3. **Flag ambiguous contributions** — If a role is unclear (e.g., someone provided feedback but not a full edit), ask whether they should be credited formally or in a general acknowledgment.

4. **Check for missing credits** — Ask explicitly: "Were any freelancers, assistants, or data sources involved?" and "Are there any unpublished studies, private communications, or off-record sources that need a note?"

5. **Assign credit placement** — For each contributor, specify where their credit belongs:
   - Byline (top of piece)
   - Contributor note (end of piece)
   - Acknowledgments section
   - Inline citation only
   - No public credit (note reason)

6. **Resolve conflicts or duplicates** — If someone appears in multiple roles, consolidate their entry and list all applicable roles together.

## Output Format

Produce a structured credit log in this format:

---

**CONTRIBUTOR CREDIT LOG**
*Piece Title:* [title]
*Publication Date / Draft Version:* [date or draft number]

---

**BYLINE**
- [Full Name] — [Role, e.g., Staff Writer]

**CONTRIBUTORS**
| Name | Role | Contribution Summary | Credit Placement |
|------|------|----------------------|-----------------|
| Jane Doe | Developmental Editor | Restructured argument in sections 2–4 | Contributor note |
| John Smith | Quoted Source | Interview on 2024-03-12; CEO of Acme Corp | Inline citation |
| … | … | … | … |

**ACKNOWLEDGMENTS SECTION DRAFT**
> [Ready-to-publish acknowledgment paragraph pulling from the log above, written in the publication's tone if specified.]

**CREDIT FLAGS / NOTES**
- [Any unresolved attributions, off-record sources, or permissions needed]
