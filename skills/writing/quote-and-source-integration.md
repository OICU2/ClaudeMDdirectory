# Quote and Source Integration

This skill takes a collection of raw quotes or source excerpts and produces a polished document where each quote is smoothly introduced, attributed, and woven into coherent prose.

## Integration Workflow

1. **Audit the raw material** — Identify every quote or excerpt, note the speaker/source/date if provided, and flag any quotes missing attribution.
2. **Establish document purpose** — Ask the user for the target audience, tone (academic, journalistic, conversational), and the central argument or theme the quotes should support. If none is given, infer from context.
3. **Cluster by theme** — Group quotes that support the same point or narrative beat. Determine the logical order: chronological, argumentative, or thematic.
4. **Write contextual lead-ins** — Precede each quote with 1–3 sentences that introduce the source's credibility, establish why the quote is relevant, and connect it to the preceding paragraph. Never drop a quote without a lead-in.
5. **Attribute precisely** — Use the format appropriate to the document style:
   - Inline prose: *According to [Name], "[quote]."* or *[Name] argues that "[quote]."*
   - Academic: parenthetical or footnote citation after the closing quotation mark.
   - Journalistic: name and title integrated into the sentence.
6. **Write follow-through sentences** — After each quote, add 1–2 sentences that interpret, connect, or advance the argument. Never let a quote be the last sentence of a paragraph.
7. **Smooth transitions** — Use transitional phrases between quote clusters to maintain narrative flow. Avoid repeating the same attribution verb more than twice in a section.
8. **Flag missing information** — If a quote lacks a source, date, or context, insert a bracketed note: *[Source needed]* or *[Date unconfirmed]* rather than inventing details.

## Output Format

Produce a single, ready-to-use document with the following structure:

- **Introductory paragraph** (3–5 sentences) that establishes the topic and signals how the sources will be used.
- **Body sections** — one section per theme cluster, each containing:
  - A topic sentence
  - One or more integrated quotes with lead-ins, attribution, and follow-through
  - A closing sentence that transitions to the next section
- **Concluding paragraph** (2–4 sentences) that synthesizes the source material without introducing new quotes.
- **Source list** at the end — a plain numbered list of all attributed sources in the order they appear, formatted as: *[Number]. [Name/Title], "[Quote snippet…]"*

Length scales to the number of quotes provided: roughly 150–200 words of original prose per 3–4 quotes. Maintain the tone the user specifies; default to clear, formal prose if no tone is given.
