# Script Writing

This skill produces ready-to-deliver scripts for video, podcast, webinar, and presentation formats — complete with pacing notes, speaker cues, and engagement hooks tailored to the target medium.

## Script Writing Workflow

1. **Identify the format** — Confirm whether the script is for video (YouTube, short-form, explainer), podcast (interview, solo, narrative), webinar (educational, sales, demo), or presentation (keynote, pitch, lecture).

2. **Clarify key parameters before writing** — Ask for: target audience, approximate runtime or word count, tone (conversational, authoritative, inspirational), and any required CTAs or key messages. If the user has provided enough context, proceed without asking.

3. **Open with a hook** — Begin every script with an attention-grabbing opening: a provocative question, surprising stat, bold statement, or relatable scenario. No slow warm-ups.

4. **Structure the body by format:**
   - **Video:** Intro hook → problem/context → solution/content → recap → CTA
   - **Podcast:** Teaser → host intro → topic setup → main discussion (with segment breaks) → key takeaways → outro/CTA
   - **Webinar:** Welcome/housekeeping → agenda preview → content sections (3–5) → Q&A bridge → close/offer
   - **Presentation:** Opening hook → thesis → supporting points (3 max) → story or example per point → conclusion → call to action

5. **Write in spoken language** — Use contractions, short sentences, and natural phrasing. Avoid jargon unless the audience demands it. Read sentences aloud mentally; if they sound stiff, rewrite them.

6. **Add speaker cues and pacing notes** — Include `[PAUSE]`, `[EMPHASIS]`, `[SLIDE ADVANCE]`, `[SHOW DEMO]`, or `[CUT TO B-ROLL]` as appropriate for the format. Mark section transitions clearly.

7. **End with a clear CTA** — Every script ends with one specific action: subscribe, download, book a call, visit a link, or apply a concept. No vague closings.

## Output Format

Produce the script as a formatted document with the following structure:

```
**SCRIPT: [Title]**
Format: [Video / Podcast / Webinar / Presentation]
Estimated Runtime: [X minutes]
Tone: [e.g., Conversational, Authoritative]
Target Audience: [Description]

---

[SECTION NAME]

[Speaker text written out in full, paragraph form]

[STAGE DIRECTION or CUE in brackets]

[Next section continues...]

---

CTA:
[Exact closing words + action]
```

- Write all speaker dialogue in full — no bullet summaries or placeholders
- Keep sections labeled and visually separated
- Runtime estimates: ~130 words per minute for conversational delivery, ~150 for energetic/sales tone
- Length scales to requested runtime; default to a 5-minute script (~650–750 words of spoken content) if no duration is specified
