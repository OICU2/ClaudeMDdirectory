# SEO Writing

Produces search-optimized written content that targets specific keywords and matches user search intent without sacrificing clarity, depth, or readability.

## SEO Writing Workflow

1. **Identify search intent** — Determine whether the target keyword signals informational, navigational, commercial, or transactional intent. Shape the content type accordingly (guide, comparison, how-to, product page).

2. **Extract keyword targets** — Identify the primary keyword, 2–4 secondary keywords, and related semantic terms (LSI keywords). If none are provided, ask the user before proceeding.

3. **Structure for featured snippets and scanability**
   - Use the primary keyword in the H1 title, naturally within the first 100 words, and in at least one H2
   - Write meta description (150–160 characters) incorporating the primary keyword early
   - Use H2s and H3s to address related subtopics and long-tail variants
   - Answer the core question directly within the first 2–3 paragraphs

4. **Write for humans first** — Paragraphs should be 2–4 sentences. Avoid keyword stuffing; target 1–2% keyword density naturally. Use active voice. Eliminate filler phrases.

5. **Add E-E-A-T signals** — Include specific data points, cite credible sources, add expert perspective or firsthand framing where appropriate. Avoid generic claims.

6. **Internal/external link placeholders** — Note where internal links to related content or authoritative external sources should go using `[LINK: description]` placeholders.

7. **Optimize for engagement signals** — Open with a hook that addresses the reader's pain point. Close with a clear CTA or next step that reduces bounce intent.

## Output Format

Produce the following in order:

**SEO Brief Summary**
- Target keyword
- Secondary keywords
- Search intent type
- Recommended content length

**Meta Title** (50–60 characters, keyword near front)

**Meta Description** (150–160 characters)

**Full Article**
- H1 matching or close to meta title
- Introduction (hook + keyword + thesis, ~100 words)
- Body sections with H2/H3 headers, bullet points or numbered lists where appropriate
- Conclusion with CTA or summary
- Minimum 600 words; scale to 1,200–2,500 for competitive or informational topics

**Keyword Usage Summary**
- List where primary and secondary keywords appear naturally in the content
