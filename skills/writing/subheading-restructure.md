# Subheading Restructure

Analyzes a draft's H1–H3 heading hierarchy for logical progression, grammatical parallelism, and keyword alignment, then delivers a restructured outline with explicit reasoning for each change.

## Audit & Restructure Workflow

1. **Extract the skeleton** — Pull all headings (H1, H2, H3) from the draft in order. If none are marked, infer section breaks from the prose.

2. **Audit for logical flow** — Check that each H2 supports the H1 argument, and each H3 narrows its parent H2. Flag any headings that are out of sequence, redundant, or orphaned from the main thesis.

3. **Audit for parallel structure** — Identify inconsistencies in grammatical form across sibling headings (e.g., mixing noun phrases with questions or gerunds at the same level). All siblings at the same level must share a consistent syntactic pattern.

4. **Audit for SEO alignment** — Identify whether the H1 and primary H2s contain the target keyword or its close variants. Flag headings that are too vague, too clever, or bury the keyword. If no target keyword is provided, infer the most likely primary topic from the content.

5. **Propose restructured outline** — Rewrite the full heading hierarchy with all issues resolved. Where headings are reordered, merged, split, or rewritten, annotate each change with a one-line reason.

6. **Surface content gaps** — List any logical subtopics implied by the restructured outline that are missing from the current draft.

## Output Format

Produce three clearly labeled sections:

**1. Audit Summary**
A bullet list of specific issues found, each tagged with its type: `[FLOW]`, `[PARALLEL]`, or `[SEO]`. Maximum 10 bullets. Be direct — name the exact heading that fails and why.

**2. Restructured Outline**
The full revised heading hierarchy in indented outline format (H1 > H2 > H3). After each changed heading, append a parenthetical note explaining the change (e.g., *(merged with next section — both covered retention tactics)*). Unchanged headings need no annotation.

**3. Content Gaps**
A numbered list of missing subtopics the restructured outline implies but the draft does not address. If none, state "No significant gaps identified."
