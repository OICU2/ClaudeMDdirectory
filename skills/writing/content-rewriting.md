# Content Rewriting

This skill rewrites existing content to meet a specific goal — improving clarity, adjusting tone, optimizing for SEO, or targeting a new audience — and Claude will use it whenever someone shares text that needs transformation rather than creation from scratch.

## Rewriting Workflow

1. **Identify the rewriting goal** — Ask if not stated: clarity, tone shift, SEO optimization, audience change, length reduction, or general improvement.
2. **Analyze the source content** — Note the current tone, structure, reading level, key messages, and any weak spots (passive voice, jargon, redundancy, unclear flow).
3. **Clarify the target audience** — Confirm who will read the rewritten version (e.g., executives, beginners, customers, search engines).
4. **Preserve core intent** — Keep the original meaning, key facts, and essential information intact unless explicitly told to change them.
5. **Apply targeted improvements based on goal:**
   - **Clarity**: Shorten sentences, eliminate jargon, use active voice, improve transitions.
   - **Tone**: Match formality level — casual, professional, empathetic, authoritative — to the stated need.
   - **SEO**: Integrate target keywords naturally, improve headings, strengthen meta-worthy opening sentences.
   - **Audience fit**: Adjust vocabulary, examples, and assumed knowledge to match the new reader.
6. **Highlight key changes** — After delivering the rewrite, briefly note what was changed and why.

## Output Format

Deliver the rewritten content in the same structural format as the original (paragraph → paragraph, bullets → bullets, heading structure preserved unless restructuring was the goal). Follow with a short **Changes Made** section (3–5 bullet points) explaining the most significant edits and the reasoning behind them. If multiple versions are appropriate (e.g., formal vs. casual), provide both with clear labels. Keep rewrites within ±20% of the original word count unless length adjustment was requested.
