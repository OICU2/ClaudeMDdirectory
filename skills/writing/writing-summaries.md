# Writing Summaries

This skill condenses long-form content into accurate, useful summaries that capture essential information, main arguments, and key details without distortion or omission of critical points.

## Summarization Workflow

1. **Identify the content type** — article, report, transcript, email thread, book chapter, etc. This shapes the summary structure.
2. **Determine the appropriate length** — default to roughly 10% of the original length unless the user specifies otherwise. Very short content gets 1–3 sentences; very long content may warrant bullet points plus a paragraph.
3. **Extract the core elements**:
   - Central argument or main purpose
   - Key facts, figures, or findings
   - Important names, dates, or entities
   - Conclusions or outcomes
4. **Preserve accuracy** — never paraphrase in a way that changes meaning, omit a finding that contradicts the main point, or inflate minor details.
5. **Match the user's purpose** — if they need an executive summary, lead with conclusions. If they need study notes, use structured bullets. If they want a quick gist, write a tight paragraph.
6. **Flag ambiguity** — if the source material is contradictory, unclear, or incomplete, note that explicitly rather than smoothing it over.

## Output Format

**Default summary structure:**

- **One-sentence gist** (what this is about, in plain language)
- **Body** (2–5 sentences or a short bullet list covering main points, key evidence, and conclusions)
- **Notable details** (optional — only if specific facts, names, or figures are critical to retain)

Length guidelines:
- Short content (under 500 words): 1–3 sentence paragraph
- Medium content (500–3,000 words): 1 paragraph or 4–6 bullets
- Long content (3,000+ words): Brief intro sentence + structured bullets + 1-sentence conclusion

Format adjustments:
- Use bullet points when the source has multiple distinct sections or findings
- Use prose when the source is narrative or argumentative
- Always match the register of the request (casual ask → plain prose, professional ask → clean structured format)
