# Content Repurposing

This skill takes existing content in any format and systematically transforms it into multiple new formats suited for different channels, audiences, and consumption contexts.

## Repurposing Workflow

1. **Analyze source content** — Identify the core message, key points, unique insights, data, quotes, and stories worth preserving across formats.

2. **Identify target formats** — If not specified, suggest 3-5 high-value formats based on the content type:
   - Long-form article → LinkedIn post, Twitter/X thread, email newsletter, short video script, infographic outline
   - Podcast/video transcript → Blog post, quote cards, key takeaways list, TikTok script
   - Research/report → Executive summary, slide deck outline, social carousel, press release
   - Case study → Testimonial snippet, how-to guide, before/after breakdown

3. **Adapt for each format** — Rewrite with format-native structure:
   - **Twitter/X thread**: Hook tweet + 5-8 numbered tweets + call-to-action
   - **LinkedIn post**: Bold opening line, short paragraphs, 3-5 key insights, engagement question
   - **Email newsletter**: Subject line + preview text + intro + 3 sections + one CTA
   - **Short video script**: Hook (0-3s), problem (3-10s), solution points (10-45s), CTA (45-60s)
   - **Carousel/slide deck**: Title slide + 5-8 content slides + summary slide
   - **Blog post**: SEO headline + intro + H2-structured body + conclusion + CTA

4. **Adjust tone and depth** — Match vocabulary and detail level to the target platform's audience expectations. LinkedIn = professional insight. Twitter = punchy and direct. Email = conversational and personal.

5. **Preserve the core value** — Every repurposed version must retain the original's primary insight or value proposition — never dilute the key message when condensing.

6. **Add format-specific hooks** — Each format needs its own attention-grabbing opener tailored to how users encounter content on that platform (scroll, inbox, search).

## Output Format

Deliver each repurposed piece as a clearly labeled section:

```
## [Format Name] — [Platform/Channel]

[Complete ready-to-publish content for that format]

Word count / character count: [X]
```

- Produce complete, publish-ready drafts — no placeholders or fill-in-the-blank gaps
- Include platform-specific metadata where relevant (e.g., suggested hashtags, email subject line, alt text suggestion)
- If repurposing into 3+ formats, add a **Repurposing Map** at the top summarizing what was created and the strategic angle for each
- Flag any content that needs updating (e.g., time-sensitive stats, broken references) with a `[UPDATE NEEDED: reason]` note inline
