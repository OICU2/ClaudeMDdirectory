# Writing Portfolio Curation

This skill analyzes a set of writing samples, organizes them into a coherent portfolio structure, and generates positioning language that helps the writer present each piece strategically to target audiences.

## Curation Workflow

1. **Inventory the samples**: Ask the user to list or paste their writing samples. For each piece, collect: title or topic, format (blog post, white paper, case study, email sequence, social copy, article, etc.), industry or niche, approximate word count, and original publication or purpose if known.

2. **Categorize by format and industry**: Group pieces into a two-axis matrix — rows are formats, columns are industries or audience types. Flag any gaps (e.g., no long-form samples, no B2B examples) that weaken the portfolio.

3. **Assess strategic value of each piece**: For every sample, evaluate:
   - Demonstrates a distinct skill (research depth, voice, persuasion, technical clarity, storytelling)
   - Fits a target client or employer type the writer is pursuing
   - Shows range vs. redundancy within the collection
   - Quality threshold: strong enough to include without apology

4. **Make include/exclude/replace recommendations**: Clearly mark each piece as INCLUDE, EXCLUDE (with reason), or STRENGTHEN (include with a caveat or revised framing). Do not pad the portfolio — a tight selection of 6–12 strong pieces outperforms a bloated 25-piece dump.

5. **Write positioning descriptions**: For each included piece, write a 2–3 sentence portfolio blurb that names the format, explains the strategic goal of the piece, and highlights the specific skill it demonstrates. Avoid vague phrases like "showcases my writing ability." Be concrete: "A 1,200-word B2B case study written for a SaaS client to move mid-funnel leads toward a demo request. Demonstrates ability to translate technical product features into outcome-focused narratives for a non-technical buyer."

6. **Propose a portfolio structure**: Suggest a presentation order and optional section headers (e.g., "Long-Form Editorial," "B2B Content Marketing," "Email & Conversion Copy") that tell a coherent story about the writer's positioning and expertise.

7. **Identify positioning gaps**: Note 1–3 types of samples the writer should create to strengthen their portfolio for their stated target market.

## Output Format

Produce the following sections in order:

**Portfolio Summary**
One paragraph (3–5 sentences) describing the writer's demonstrated strengths, range, and recommended positioning based on the samples provided.

**Curated Portfolio Structure**
A structured list organized by section headers. Under each section, list the included pieces with:
- Title/description
- Format and industry tag
- Positioning blurb (2–3 sentences)
- INCLUDE / STRENGTHEN label (with note if strengthening is needed)

**Excluded Pieces**
A brief list of excluded samples with a one-line reason for each.

**Gap Analysis**
A bulleted list of 1–3 missing sample types the writer should produce, with a sentence explaining why each would strengthen their positioning.

Total output length should be proportional to the number of samples provided. For a 10-piece collection, expect 600–900 words of output. Use headers, labels, and bullet points — no dense prose blocks.
