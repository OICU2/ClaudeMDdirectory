# Content Localization and Cultural Adaptation

This skill transforms existing written content to feel authentically native to a target regional audience by systematically replacing idioms, cultural references, spelling conventions, date/number formats, and tone patterns with locally appropriate equivalents.

## Localization Workflow

1. **Identify the target region**: Confirm the specific locale (e.g., UK English, Australian English, Brazilian Portuguese, Canadian French). If not specified, ask before proceeding.

2. **Audit the source content** for these localization targets:
   - Spelling conventions (e.g., color → colour, organize → organise for UK; -ize is preferred in AU academic contexts)
   - Idioms and expressions (e.g., "ballpark figure" → "rough estimate" for non-US audiences; "touch wood" vs. "knock on wood")
   - Cultural references (sports, celebrities, TV shows, political figures, holidays)
   - Units of measurement (miles → kilometres, Fahrenheit → Celsius, pounds → kilograms)
   - Date and number formats (MM/DD/YYYY → DD/MM/YYYY; 1,000.50 → 1.000,50 for many European locales)
   - Currency symbols and conventions
   - Formality and tone register (UK English often more formal; Australian English more casual and direct)
   - Local legal or regulatory terminology if relevant

3. **Replace systematically**: Go through each category above and substitute all instances. Do not leave partial localizations.

4. **Preserve meaning and intent**: Every adapted element must convey the same meaning as the original. Flag any references that have no clean local equivalent and suggest alternatives.

5. **Adjust tone register**: Match the cultural communication style of the target region. Australians expect directness and humor; Germans expect precision and formality; UK audiences often appreciate understatement.

6. **Flag unresolvable references**: If a cultural reference (e.g., a specific US political event) has no meaningful local equivalent, note it explicitly and offer two options: remove it or replace it with a region-specific equivalent.

## Output Format

Produce two clearly labeled sections:

**Localized Content**
The fully adapted version of the text, ready to publish. No brackets, no placeholders, no editorial notes embedded in the text.

**Localization Notes**
A bulleted list of every change made, grouped by category (Spelling, Idioms, Cultural References, Formatting, Tone). Each bullet should read: `[Original] → [Adapted] — [brief reason]`. Flag any items that required a judgment call or had no perfect equivalent.

If the input content is longer than ~500 words, process it in full but summarize the Localization Notes by category count (e.g., "12 spelling changes, 4 idiom replacements") with only notable or ambiguous changes listed individually.
