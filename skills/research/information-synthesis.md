# Information Synthesis

This skill merges data, research, or content from multiple sources into a unified, structured analysis that highlights consensus, conflicts, and actionable conclusions.

## Synthesis Workflow

1. **Inventory the sources** — List each source briefly (type, origin, scope) before doing any analysis.
2. **Extract core claims** — Pull the 2–5 most significant claims or data points from each source.
3. **Find convergence** — Identify where sources agree; treat overlapping points as high-confidence findings.
4. **Surface contradictions** — Flag where sources conflict. Note which source is more recent, more rigorous, or more relevant to the user's context.
5. **Fill gaps** — Identify what the combined sources still leave unanswered or ambiguous.
6. **Draw conclusions** — Synthesize a set of integrated findings that could not come from any single source alone.
7. **Generate action items** — Translate findings into concrete next steps, decisions, or recommendations relevant to the user's goal.

## Output Format

Produce a structured synthesis with these sections:

**Sources Reviewed** — Bullet list of sources with a one-line description each.

**Key Findings** — Numbered list of 3–7 unified insights drawn across sources. Each insight should be one to two sentences and cite which sources support it.

**Conflicts & Caveats** — Bulleted list of contradictions or uncertainties, with a brief note on how to interpret or resolve each.

**Gaps** — Two to four sentences on what the sources collectively fail to address.

**Recommended Actions** — Numbered list of 3–5 concrete next steps based on the synthesis.

Keep the total response scannable. Use plain language. Avoid restating source content verbatim — synthesize, don't summarize.
