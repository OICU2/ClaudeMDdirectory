# Research Findings Contradiction Resolution

This skill maps out contradictions between research sources, diagnoses the most probable causes, and recommends actionable paths to resolution whenever conflicting evidence is encountered.

## Contradiction Analysis Workflow

1. **Catalog the conflict precisely.** State each contradictory finding in one sentence, naming the source, the claim, and the direction of the discrepancy (e.g., Study A finds positive effect, Study B finds null effect).

2. **Extract study characteristics for comparison.** For each conflicting source, document: sample size and population, study design (RCT, observational, meta-analysis, etc.), measurement instruments and operationalization of key variables, time period and geographic context, and funding source or potential conflicts of interest.

3. **Identify explanatory factors.** Systematically check for these common contradiction drivers:
   - **Methodological differences** — different controls, statistical thresholds, or analytical choices
   - **Population heterogeneity** — findings may both be true for different subgroups
   - **Temporal or contextual shifts** — conditions changed between studies
   - **Publication bias** — one direction of finding is overrepresented
   - **Operationalization gaps** — the "same" variable is measured differently
   - **Effect size vs. significance confusion** — statistical vs. practical significance differ

4. **Assign likelihood to each explanatory factor.** Rate each factor as High / Medium / Low based on evidence in the sources. Do not speculate without grounding.

5. **Recommend resolution approaches.** Choose from:
   - Accept the finding from the higher-quality or more applicable study and explain why
   - Treat findings as conditionally true (each applies under specific conditions)
   - Flag as genuinely unresolved and specify what additional research would settle it
   - Recommend a meta-analysis or systematic review if one does not already exist

6. **State a working conclusion.** Give a single actionable takeaway the user can act on now, with explicit confidence level (high / moderate / low) and the primary caveat.

## Output Format

Produce a structured report with these sections:

**Contradiction Summary**
- One-paragraph description of what conflicts and why it matters

**Source Comparison Table**
| Factor | Source A | Source B |
|---|---|---|
| Claim | | |
| Design | | |
| Sample | | |
| Measurement | | |
| Key limitations | | |

**Explanatory Factors**
- Bulleted list, each factor labeled High/Medium/Low likelihood with one-sentence rationale

**Resolution Recommendation**
- Numbered steps or a single clear directive depending on complexity

**Working Conclusion**
- 2–3 sentences: what to conclude now, confidence level, and what would change the conclusion

Length: 400–700 words total. Use plain language. Avoid academic hedging that obscures the recommendation.
