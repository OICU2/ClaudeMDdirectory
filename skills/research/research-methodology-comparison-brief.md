# Research Methodology Comparison Brief

This skill generates a structured, side-by-side evaluation of two or more research methodologies against user-defined or inferred criteria to help researchers select the most appropriate approach for their study.

## Methodology Comparison Workflow

1. **Identify the research question and context** — Extract or ask for the core research question, study population, available resources, and any constraints (budget, timeline, ethics, access).

2. **Identify candidate methodologies** — List the methodologies the user is considering. If none are specified, propose 2–4 relevant options based on the research question (e.g., survey, interviews, randomized controlled trial, ethnography, secondary data analysis, mixed methods).

3. **Define evaluation criteria** — Use these standard criteria unless the user specifies others:
   - **Validity** (internal and external)
   - **Reliability**
   - **Cost** (low / medium / high)
   - **Timeline** (weeks / months / years)
   - **Sample size requirements**
   - **Ethical complexity**
   - **Fit to research question**
   - **Researcher skill requirements**

4. **Score or rate each methodology per criterion** — Use a consistent scale: High / Medium / Low, or a 1–5 rating with a brief justification for each cell. Do not leave cells empty.

5. **Write a synthesis paragraph** — After the table, summarize which methodology is the strongest fit and why, noting any critical trade-offs or conditions that could change the recommendation.

6. **Flag open questions** — List 2–3 questions the researcher should answer before finalizing their choice (e.g., "Do you have IRB approval capacity?" or "Is your target population accessible for in-depth interviews?").

## Output Format

Produce the following structure in this order:

**Research Question (restated):** One sentence confirming the question being addressed.

**Methodologies Under Review:** Bulleted list of the methods being compared.

**Comparison Table:**
| Criterion | [Method A] | [Method B] | [Method C] |
|---|---|---|---|
| Validity | | | |
| Reliability | | | |
| Cost | | | |
| Timeline | | | |
| Sample Size Needed | | | |
| Ethical Complexity | | | |
| Fit to Research Question | | | |
| Skill Requirements | | | |

Each cell must contain a rating (High/Medium/Low or 1–5) plus a 5–15 word justification.

**Synthesis (2–4 sentences):** Recommend the strongest methodology and explain the key trade-offs.

**Open Questions Before Deciding (2–3 bullets):** Specific questions the researcher must resolve.

Total length: 400–700 words. Use plain language. Avoid jargon unless the user has used it first.
