# Industry Analysis

Conducts structured industry analysis using established frameworks to map competitive forces, trends, and market dynamics whenever a user wants to understand how an industry operates or assess its opportunity.

## Analysis Workflow

1. **Define the industry scope** — Clarify the specific industry, sub-sector, and geographic boundaries before proceeding. If ambiguous, state assumptions explicitly.

2. **Apply Porter's Five Forces** — Systematically assess each force:
   - Threat of new entrants (barriers to entry, capital requirements, regulation)
   - Bargaining power of suppliers (concentration, switching costs, alternatives)
   - Bargaining power of buyers (concentration, price sensitivity, substitutes)
   - Threat of substitutes (availability, performance, switching costs)
   - Competitive rivalry (number of players, differentiation, exit barriers)

3. **Identify macro trends** — Map 3–5 forces reshaping the industry using PESTLE lens (Political, Economic, Social, Technology, Legal, Environmental). Focus only on trends with material impact.

4. **Assess value chain structure** — Identify where value is created and captured. Note which segments are commoditized vs. differentiated. Flag where margins concentrate.

5. **Evaluate industry attractiveness** — Synthesize findings into an overall attractiveness rating (High / Medium / Low) with explicit reasoning tied to the forces and trends above.

6. **Surface key uncertainties** — List 2–3 critical unknowns that could materially change the analysis (regulatory shifts, technology disruption, demand shocks).

## Output Format

Produce a structured report with the following sections:

**Industry Snapshot** — 2–3 sentences defining the industry, size estimate if available, and stage (emerging / growth / mature / declining).

**Five Forces Assessment** — One paragraph per force. Each paragraph states the force strength (Strong / Moderate / Weak), the primary drivers, and a concrete example or data point.

**Key Trends** — Bulleted list of 3–5 trends. Each bullet: trend name in bold, one sentence on what it is, one sentence on its strategic implication.

**Value Chain Summary** — Short prose (3–5 sentences) describing where power and margin sit in the chain.

**Attractiveness Rating** — Single-line verdict (High / Medium / Low) followed by 3–4 sentences of synthesis.

**Key Uncertainties** — Bulleted list of 2–3 items, each with a brief explanation of the potential impact.

Total length: 600–900 words. Use headers exactly as described. Avoid jargon without explanation. Cite specific companies, regulations, or data points wherever possible to ground the analysis.
