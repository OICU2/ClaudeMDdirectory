# Research Sample Stratification Plan

This skill creates a complete stratified sampling framework — defining strata boundaries, proportional targets, and recruitment criteria — whenever a researcher needs to structure how they will draw a representative sample from a study population.

## Stratification Workflow

1. **Clarify the research population**: Identify the full target population (e.g., adults with Type 2 diabetes in urban clinics). If not stated, ask for inclusion/exclusion criteria before proceeding.

2. **Identify stratification variables**: Determine which variables are theoretically or practically important to control (e.g., age group, sex, disease severity, geographic region). Limit to 2–4 variables to avoid over-stratification.

3. **Define strata boundaries**: For each variable, specify discrete, mutually exclusive categories with clear boundary rules (e.g., Age: 18–34, 35–54, 55–74, 75+). Flag any overlapping or ambiguous boundaries.

4. **Determine sampling method per stratum**: Choose proportional allocation (mirrors population distribution), disproportional allocation (oversamples rare groups), or optimal allocation (weights by variance or cost). State the rationale.

5. **Calculate proportional targets**: Using known or estimated population proportions, calculate the target n per stratum for the total desired sample size. Show the math explicitly (e.g., stratum proportion × total N = stratum n).

6. **Define recruitment criteria per stratum**: For each stratum, list specific eligibility criteria, screening questions, and any stratum-specific exclusions.

7. **Address practical constraints**: Flag strata that are likely hard to recruit (small populations, stigmatized groups, geographic barriers) and recommend oversampling buffers (typically 20–30% above target) or alternative recruitment channels.

8. **Specify replacement rules**: Define criteria for replacing a participant who drops out or is found ineligible post-enrollment within a stratum.

## Output Format

Produce a structured stratification plan with the following sections:

**1. Population Definition**
- Full target population statement
- Inclusion/exclusion criteria

**2. Stratification Variables**
- Table listing each variable, its categories, and boundary definitions

**3. Stratum Matrix**
- A grid or table showing all strata combinations (if crossing multiple variables), with a unique stratum ID for each cell

**4. Sample Allocation Table**
| Stratum ID | Description | Population % | Target n | Buffer n (if applicable) |
|---|---|---|---|---|

**5. Recruitment Criteria per Stratum**
- Bullet list per stratum: eligibility rules, screening questions, exclusions

**6. Sampling Method Justification**
- 2–4 sentences explaining why proportional, disproportional, or optimal allocation was chosen

**7. Risk Flags & Recommendations**
- Bulleted list of hard-to-reach strata and mitigation strategies

Length: Comprehensive but scannable. Use tables wherever data is comparative. Avoid prose where a table or list serves better.
