# Research Documentation

This skill produces a structured research document capturing methodology, sources, and findings so work is reproducible, referenceable, and shareable.

## Documentation Workflow

1. **Capture the research question** — State the exact question or hypothesis being investigated, including scope and any constraints.
2. **Record methodology** — Document the search strategy, tools used (databases, search engines, APIs), date of research, and any inclusion/exclusion criteria applied to sources.
3. **Log all sources** — For each source, record: title, author(s), publication date, URL or DOI, access date, and a one-line relevance note.
4. **Summarize key findings** — Extract the main findings from each source, noting agreements, contradictions, and gaps across sources.
5. **Assess source quality** — Rate each source by credibility (peer-reviewed, primary, secondary, opinion) and flag any bias or limitations.
6. **Synthesize conclusions** — Write a summary that directly answers the research question, supported by the documented evidence.
7. **Note open questions** — List unresolved questions, areas needing further research, and recommended next steps.

## Output Format

Produce a Markdown document with the following sections:

```
# Research Document: [Topic]

**Research Question:** [Exact question]
**Researcher:** [Name if provided]
**Date:** [Date of research]
**Status:** [Draft / Complete / Ongoing]

## Methodology

- Search terms/queries:
- Date range of sources:
- Inclusion criteria:
- Exclusion criteria:

## Sources

### Source 1
- **Title:**
- **Author(s):**
- **Published:**
- **URL/DOI:**
- **Accessed:**
- **Type:** [Peer-reviewed / Primary / Secondary / Opinion]
- **Relevance:**
- **Key points:**

[Repeat for each source]

## Findings Summary
[2–4 paragraphs synthesizing findings across all sources]

## Conclusions
[Direct answer to the research question with supporting evidence]

## Gaps & Open Questions
- [Unresolved question 1]
- [Unresolved question 2]

## Recommended Next Steps
- [Action 1]
- [Action 2]
```

Length scales with research complexity — a focused question may produce 300–500 words; a broad topic may produce 1,000+. Every section must be completed; use "Not determined" rather than omitting a field.
