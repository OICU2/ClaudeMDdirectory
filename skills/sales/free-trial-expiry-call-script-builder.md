# Free Trial Expiry Call Script Builder

This skill generates a personalized, structured call script for sales reps to convert expiring or expired free trial users by surfacing usage gaps, addressing conversion barriers, and creating urgency around a clear next step.

## Script Building Workflow

1. **Gather context before generating** — Ask for the following if not provided:
   - Prospect name, company, and role
   - Product or service name
   - Trial expiry date (days remaining or already expired)
   - Any known usage data (features used, login frequency, team size)
   - Known objections or hesitations (if any)
   - Desired outcome of the call (demo, upgrade, extension, contract)

2. **Open with a warm, specific hook** — Reference the trial directly and personalize with a known usage detail or company context. Avoid generic openers. Example anchor: "I noticed you've been active in [Feature X] but haven't had a chance to explore [Feature Y]."

3. **Diagnose before pitching** — Build 3–4 discovery questions that uncover:
   - What the prospect was hoping to solve during the trial
   - Where they got stuck or disengaged
   - Who else is involved in the buying decision
   - What success would look like post-purchase

4. **Bridge usage gaps to value** — Map any underused features or low engagement to missed outcomes. Frame the upgrade as completing what the trial started, not starting something new.

5. **Surface and pre-handle objections** — Include 2–3 common objection responses tailored to the product category:
   - Price / budget timing
   - "We need more time to evaluate"
   - "I need to get buy-in from my team"

6. **Create urgency without pressure** — Reference the trial expiry date naturally. Offer a concrete incentive if applicable (extended trial, onboarding support, pricing lock). Avoid manufactured scarcity.

7. **Close with a single clear next step** — End every script with one specific ask: book a follow-up call, start a contract, schedule an onboarding session. Never leave the call with an ambiguous close.

8. **Include a voicemail fallback** — Provide a 20–30 second voicemail script if the prospect doesn't pick up, referencing the expiry date and a specific callback hook.

## Output Format

Produce a complete call script with the following labeled sections:

- **Pre-Call Notes** (2–3 bullets summarizing prospect context and call goal)
- **Opening (0:00–0:30)** — Word-for-word opener with personalization placeholder tags (e.g., `[PROSPECT NAME]`, `[FEATURE USED]`)
- **Discovery Questions** — Numbered list of 3–4 questions with brief notes on what each uncovers
- **Value Bridge** — 2–4 sentences connecting their trial behavior to the paid outcome
- **Objection Responses** — 2–3 objections with concise, natural-language rebuttals (label each objection clearly)
- **Close** — Word-for-word closing ask with one specific CTA
- **Voicemail Script** — Standalone 20–30 second script, clearly separated

Total script length: 400–700 words. Use plain conversational language. No corporate jargon. Format for easy reading during a live call — short paragraphs, clear labels, no dense blocks of text.
