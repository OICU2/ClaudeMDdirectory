# Sales Scripts

Generates structured, conversion-focused sales scripts for any channel — cold calls, outreach emails, or live demos — tailored to the product, audience, and goal.

## Script Creation Workflow

1. **Identify the channel** — Determine if the script is for a cold call, follow-up call, cold email, follow-up email, or demo talk track. Each has a distinct structure.
2. **Clarify the core offer** — Extract what is being sold, the primary value proposition, and the ideal customer profile (ICP). Ask if not provided.
3. **Define the goal** — Is the script meant to book a meeting, close a deal, re-engage a cold lead, or guide a live demo? The CTA depends on this.
4. **Build the script using proven frameworks:**
   - **Cold calls:** Pattern interrupt opener → credibility statement → pain-focused question → value bridge → soft CTA
   - **Cold emails:** Subject line → one-sentence hook → social proof or pain point → value proposition → single clear CTA (under 150 words)
   - **Demo talk tracks:** Discovery recap → agenda setting → problem-solution narrative per feature → ROI framing → objection anticipation → next steps close
5. **Embed objection handling** — Include 2–3 likely objections with short, confident responses directly in the script where they naturally arise.
6. **Write for spoken or scanned delivery** — Calls use short sentences and natural pauses. Emails use white space and bold key phrases. Demos use signpost language ("Now let me show you...").
7. **End with a definitive CTA** — Every script closes with one specific, low-friction next step (schedule a call, reply to this email, confirm next steps).

## Output Format

Produce a complete, ready-to-use script with the following structure:

**Header:**
- Channel type (Cold Call / Email / Demo)
- Target audience / ICP
- Primary goal / CTA

**Script body:**
- Clearly labeled sections (e.g., `[OPENER]`, `[VALUE PROP]`, `[OBJECTION: Price]`, `[CLOSE]`)
- Spoken scripts formatted with natural line breaks and stage directions in brackets (e.g., `[pause]`, `[wait for response]`)
- Emails include subject line variants (2–3 options)

**Objection block (calls and demos):**
- Format: **Objection:** "..." → **Response:** "..."
- Include 2–3 objections minimum

**Length:**
- Cold call script: 1–2 minutes of spoken content (~200–300 words)
- Cold email: under 150 words body + 3 subject line options
- Demo talk track: full flow with timing estimates per section (total 20–45 min)
