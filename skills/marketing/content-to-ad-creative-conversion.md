# Content to Ad Creative Conversion

This skill extracts the highest-value messaging from long-form content and structures it into actionable ad creative briefs with headlines, hooks, body copy, and visual direction for Facebook/Instagram, Google, LinkedIn, and TikTok/Reels formats.

## Conversion Workflow

1. **Analyze the source content**
   - Identify the core value proposition (the single most important benefit or insight)
   - Extract 3–5 supporting proof points, stats, or compelling claims
   - Note the target audience implied by the content's language and framing
   - Flag any emotionally resonant phrases, vivid examples, or quotable sentences

2. **Identify ad angles**
   - Derive at least 3 distinct angles from the content:
     - **Problem-aware**: leads with pain point the content addresses
     - **Curiosity/Intrigue**: leads with the most surprising or counterintuitive insight
     - **Social proof or outcome**: leads with a result, stat, or transformation
   - Each angle must map to a specific section or claim from the source content

3. **Build platform-specific creative briefs**
   For each angle, produce briefs tailored to:
   - **Facebook/Instagram Feed** (image or carousel): headline + primary text + CTA
   - **Google Search**: 3 headlines (30 chars each) + 2 descriptions (90 chars each)
   - **LinkedIn Sponsored Content**: hook line + 3-sentence body + CTA suited to professional context
   - **TikTok/Reels/Short Video**: 3-second spoken hook + key talking point sequence (5–7 bullet points) + CTA

4. **Write the copy**
   - Headlines: lead with the benefit or hook, not the brand name
   - Hooks: first line must create tension, curiosity, or immediate relevance
   - Body copy: one idea per sentence, active voice, no jargon unless present in source
   - CTAs: match intent (learn → "Read the full story," convert → "Get started free," download → "Grab the guide")

5. **Add visual direction**
   - Suggest a visual concept for each platform brief (photo style, illustration, text overlay, or video treatment)
   - Visual direction must directly reflect the ad angle, not be generic

## Output Format

Produce a structured brief document with the following sections:

---

**Source Summary**
- Core value proposition: [1 sentence]
- Target audience: [1–2 sentences]
- Top proof points: [bulleted list of 3–5 items]

---

**Ad Angle 1: [Angle Name]**

*Facebook/Instagram*
- Headline: [text]
- Primary Text: [2–4 sentences]
- CTA: [button text]
- Visual Direction: [1–2 sentences]

*Google Search*
- Headline 1 / Headline 2 / Headline 3
- Description 1 / Description 2

*LinkedIn*
- Hook: [1 sentence]
- Body: [3 sentences]
- CTA: [text]
- Visual Direction: [1 sentence]

*TikTok/Reels*
- 3-Second Hook (spoken): [text]
- Talking Points: [5–7 bullets]
- Closing CTA: [text]

---

Repeat the full block for Angle 2 and Angle 3.

Total output length: approximately 600–900 words depending on content complexity. Use headers and clear labels throughout so the brief can be handed directly to a designer or media buyer.
