# Competitive Ad Analysis

This skill systematically breaks down competitor ad creatives, messaging frameworks, and platform strategies to surface actionable gaps and positioning opportunities for differentiation.

## Analysis Workflow

1. **Identify Competitors**
   - Confirm the target brand and list direct, indirect, and aspirational competitors to analyze
   - Clarify which platforms to cover (Meta, Google, TikTok, LinkedIn, YouTube, programmatic)
   - Ask for any known competitor ad examples or ad library links if available; otherwise work from provided descriptions or publicly known campaigns

2. **Extract Messaging Patterns**
   - Identify the core value proposition each competitor leads with
   - Note recurring hooks, headlines, and calls-to-action
   - Classify messaging by theme: price/value, emotional appeal, feature-led, social proof, urgency, identity/aspiration
   - Flag which claims are generic vs. ownable (e.g., "high quality" vs. a specific proof point)

3. **Assess Creative Strategy**
   - Identify dominant formats used per competitor (static image, video, carousel, UGC-style, testimonial)
   - Note visual tone: polished vs. raw, aspirational vs. functional, brand-heavy vs. product-forward
   - Identify talent/spokesperson patterns: founder, customer, actor, no talent
   - Note copy length patterns: short punchy vs. long-form educational

4. **Map Positioning**
   - Build a 2x2 or axis map of how competitors cluster (e.g., premium vs. accessible, emotional vs. rational)
   - Identify which positioning territory is overcrowded
   - Identify which positioning territory is underserved or uncontested

5. **Identify Gaps and Opportunities**
   - List claims no competitor is making that the target brand could credibly own
   - List formats or tones no competitor is using that could create pattern interruption
   - List audience segments or pain points competitors are ignoring
   - Prioritize opportunities by: credibility fit, competitive whitespace, and platform relevance

6. **Synthesize Recommendations**
   - Provide 3–5 specific differentiation angles the target brand should test
   - For each angle, specify: the core message, suggested format, suggested platform, and why it's differentiated
   - Flag any competitor messaging the target brand should avoid because it is saturated or legally risky

## Output Format

Produce a structured report with these sections:

**Competitor Overview Table**
| Competitor | Primary Platform | Lead Value Prop | Dominant Format | Tone |
|---|---|---|---|---|
(one row per competitor)

**Messaging Pattern Summary**
- Bullet list of recurring themes across competitors with frequency notes

**Positioning Map**
- Describe the axes and where each competitor sits (text-based if visual not possible)
- Call out the white space explicitly

**Gap Analysis**
- 3–6 bullet points identifying uncontested messaging, format, or audience opportunities

**Differentiation Recommendations**
- Numbered list of 3–5 recommended angles, each with: Message / Format / Platform / Why It Wins

Keep the full output between 600–1,000 words. Use plain language. No jargon unless it is standard marketing terminology.
