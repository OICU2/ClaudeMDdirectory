# Social Media Bio Optimization Audit

This skill audits social media bios across one or more platforms to identify inconsistencies, missing keywords, and weak calls-to-action, then delivers prioritized recommendations to improve discoverability and profile conversion.

## Audit Workflow

### 1. Collect Bio Content
Request the bio text from each platform being audited (Instagram, LinkedIn, Twitter/X, TikTok, Facebook, YouTube, etc.). If the user only provides one platform, flag that a cross-platform audit would strengthen results and proceed with what's given.

### 2. Identify the Core Brand Promise
Extract the single clearest value proposition from the bios provided. If none is evident, flag this as a critical gap. Ask: Does the bio answer "Who I/we are, who I/we serve, and what I/we do" within the first two lines?

### 3. Run the Consistency Check
Compare all bios against these five consistency dimensions:
- **Name/Handle format**: Are they identical or intentionally adapted per platform norms?
- **Core value proposition**: Is the same benefit communicated across all bios, even if phrasing differs?
- **Tone and voice**: Does the personality match across platforms, or does it shift in ways that feel disconnected?
- **Keywords and niche terms**: Are the same audience-relevant keywords present on each platform?
- **Contact/link strategy**: Is the primary CTA destination consistent, or do bios point to conflicting destinations?

### 4. Run the Keyword Gap Analysis
Identify missing high-value keywords by:
- Noting what industry, niche, or audience terms are absent
- Checking whether platform-specific searchable terms are used (e.g., LinkedIn uses job titles and skills for search; Instagram uses niche hashtag-adjacent terms in bios)
- Flagging generic descriptors ("entrepreneur," "lover of life") that consume space without aiding discoverability

### 5. Evaluate Calls-to-Action
Score each bio's CTA on these criteria:
- **Present**: Is there a CTA at all?
- **Specific**: Does it tell the user exactly what to do ("Download the free guide" vs. "Check the link")?
- **Urgent or benefit-driven**: Does it give a reason to act now?
- **Linked**: Is there a functioning link or next step provided where the platform allows?

### 6. Flag Platform-Specific Violations
Apply platform rules:
- **Instagram**: 150-char limit, line breaks matter, link in bio is the only clickable link — CTA must reflect this
- **LinkedIn**: Headline (220 chars) and About section are separate — both need optimization; keywords drive search ranking
- **Twitter/X**: 160-char limit, pinned tweet can extend the CTA — flag if unused
- **TikTok**: 80-char limit, link only available to business accounts — flag if personal account is missing this upgrade opportunity
- **YouTube**: First 100 chars of About section appear in search — front-load value proposition

### 7. Prioritize Findings
Classify every finding as:
- 🔴 **Critical**: Directly hurts conversion or discoverability (no CTA, conflicting brand message, missing keywords)
- 🟡 **Important**: Reduces effectiveness but not immediately damaging (weak CTA, tone inconsistency)
- 🟢 **Enhancement**: Marginal gains (formatting, emoji use, link tracking)
