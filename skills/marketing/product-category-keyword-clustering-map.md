# Product Category Keyword Clustering Map

This skill analyzes a list of target keywords and organizes them into semantic clusters aligned to specific product categories and funnel stages (awareness, consideration, decision) to power content planning and paid search account structure.

## Clustering Workflow

1. **Collect input**: Ask for the keyword list (raw or with volume/CPC data), the product catalog or category names, and the primary goal (SEO content, PPC structure, or both).

2. **Identify semantic themes**: Scan keywords for shared modifiers, intent signals, and topic overlap. Group keywords that target the same concept, problem, or product feature — even if phrasing differs.

3. **Map clusters to product categories**: Assign each semantic cluster to the most relevant product category. If a cluster spans multiple categories, flag it as a cross-category opportunity.

4. **Assign funnel stage** to each cluster using these rules:
   - **Awareness**: Informational queries, how-to, what-is, problem-description terms
   - **Consideration**: Comparison, best-of, reviews, alternatives, features terms
   - **Decision**: Brand + product terms, buy/price/discount/near-me, high-purchase-intent modifiers

5. **Identify gaps**: Note product categories with no keyword coverage in a given funnel stage.

6. **Flag priority clusters**: Mark clusters with high combined search volume or strong commercial intent as high priority.

## Output Format

Produce a structured table followed by a gap analysis summary.

**Keyword Cluster Map Table**

| Cluster Name | Sample Keywords (3–5) | Product Category | Funnel Stage | Est. Volume | Priority |
|---|---|---|---|---|---|
| Running Shoe Cushioning | "best cushioned running shoes", "high cushion trainers" | Footwear – Running | Consideration | 18,000/mo | High |

Include one row per cluster. Aim for 5–20 clusters depending on keyword list size.

**Gap Analysis (bullet list)**
- List each product category that is missing Awareness, Consideration, or Decision coverage
- Note any clusters that don't map cleanly to a single category
- Recommend 2–3 content or campaign actions to close the most critical gaps

**Recommended Ad Group or Content Hub Names**
- Provide one suggested name per cluster formatted for direct use in Google Ads or a CMS content hub
