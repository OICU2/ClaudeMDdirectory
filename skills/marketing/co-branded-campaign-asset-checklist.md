# Co-Branded Campaign Asset Checklist

This skill generates a structured asset checklist and brand compliance review framework to help teams coordinate deliverables, enforce brand standards, and execute co-branded marketing campaigns cleanly with external partners.

## Workflow

### Step 1: Gather Campaign Context
Before generating the checklist, collect the following details (ask the user if not provided):
- Names of both brands involved
- Campaign name or theme
- Campaign type (product launch, event, content series, paid media, etc.)
- Key channels (email, social, paid ads, landing page, OOH, print, etc.)
- Timeline and launch date
- Who owns final brand approval on each side

### Step 2: Build the Shared Asset Checklist
Organize all required deliverables by channel. For each asset, include:
- **Asset name** (e.g., "Co-branded Instagram Story — static")
- **Dimensions/specs** required
- **Owner** (Brand A, Brand B, or shared)
- **Due date**
- **Status** (Not Started / In Progress / In Review / Approved / Final)
- **File format required** (PNG, MP4, PDF, etc.)
- **Notes** (usage rights, print vs. digital, versioning instructions)

Standard asset categories to cover:
- **Digital Paid Media**: Display banners (all IAB sizes), social static and video ads, YouTube pre-roll
- **Organic Social**: Feed posts, Stories, Reels/Shorts, LinkedIn graphics
- **Email**: Header banners, CTA buttons, footer lockups
- **Web**: Landing page hero, OG image, favicon if applicable
- **Sales/Event Collateral**: One-pagers, sell sheets, event signage, booth graphics
- **PR & Comms**: Press release header, media kit cover, spokesperson headshots
- **Internal**: Campaign brief, talking points, FAQ document

### Step 3: Apply the Brand Compliance Review Framework
For each asset produced by either party, enforce the following review checklist:

**Logo Usage**
- [ ] Both logos present and meet minimum size requirements
- [ ] Logo clear space rules respected for both brands
- [ ] No logo distortion, recoloring, or unapproved lockup
- [ ] Logo hierarchy agreed upon and applied consistently (primary/secondary placement)
- [ ] Co-brand lockup file sourced from approved assets, not recreated

**Typography**
- [ ] Brand-approved typefaces used for each brand's owned copy elements
- [ ] Fallback fonts specified for digital/web contexts
- [ ] Type sizes meet accessibility minimums (minimum 12pt print, 16px web body)

**Color**
- [ ] Brand color palettes applied correctly (CMYK for print, HEX/RGB for digital)
- [ ] No unauthorized color combinations between the two brand palettes
- [ ] Sufficient contrast ratios met (WCAG AA minimum)

**Imagery & Photography**
- [ ] All images licensed or owned — usage rights documented
- [ ] Imagery style aligns with both brand guidelines
- [ ] No competitor products, messaging, or imagery present

**Copy & Messaging**
- [ ] Brand voice and tone guidelines respected for each brand's written elements
- [ ] Legal disclaimers, trademark symbols (™, ®), and copyright lines included
- [ ] Claims reviewed and approved by legal/compliance if required
- [ ] Both brand taglines or boilerplate copy approved by respective teams
