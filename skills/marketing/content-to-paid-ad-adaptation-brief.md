# Content-to-Paid Ad Adaptation Brief

This skill transforms an existing organic content piece (blog post, video, newsletter, social post, etc.) into structured creative briefs for paid ad variants across multiple platforms, formats, and audience segments.

## Workflow

### Step 1: Extract Core Content Elements
- Identify the primary message, argument, or value proposition in the source content
- Pull 3–5 supporting proof points, stats, quotes, or hooks
- Note the content's existing tone (educational, entertaining, inspirational, promotional)
- Identify any CTAs, offers, or conversion goals implied or stated

### Step 2: Define Audience Segments
- Derive 2–3 distinct audience segments who would respond to this content (e.g., awareness-stage prospects, retargeting audience, existing customers, specific demographics or job titles)
- For each segment, note their likely pain point, motivation, and awareness level

### Step 3: Map to Ad Formats
For each audience segment, produce briefs for the following formats (adjust based on platforms specified):
- **Short-form text ad** (Google Search / LinkedIn Text): Headline (30 chars), Description (90 chars), CTA
- **Social feed ad** (Facebook/Instagram/LinkedIn): Primary text (125 chars), Headline (27 chars), Link description, CTA, Visual direction note
- **Long-form native/story ad**: Hook sentence, body (50–75 words), CTA
- **Video script outline**: Hook (0–3 sec), Core message (3–25 sec), CTA (25–30 sec)

### Step 4: Write Variant Copy
- Write 2 headline/hook variants per format per segment (angle A: benefit-led, angle B: problem/pain-led)
- Keep the original content's credibility signals (data, quotes, author authority) where they fit
- Adjust reading level and tone to match each segment's awareness level
- Flag any claims that require legal/compliance review with [REVIEW NEEDED]

### Step 5: Add Testing Recommendations
- Recommend which variant to prioritize for each segment and why
- Suggest one A/B test hypothesis per format
- Note any assets (images, video clips, landing pages) needed from the source content

## Output Format

Produce a brief document with the following structure:

```
## Ad Adaptation Brief: [Content Title]
Source: [URL or title of original content]
Goal: [Awareness / Traffic / Leads / Conversions]

---

### Audience Segment 1: [Name]
Pain point / motivation: [1 sentence]
Awareness level: [Cold / Warm / Hot]

#### Google Search Ad
- Headline A: ...
- Headline B: ...
- Description: ...
- CTA: ...

#### Social Feed Ad (Facebook/Instagram)
- Primary text A: ...
- Primary text B: ...
- Headline: ...
- Visual direction: ...
- CTA: ...

#### Video Script Outline
- Hook: ...
- Core message: ...
- CTA: ...

---

### Audience Segment 2: [Name]
[Same structure]

---

### Testing Recommendations
- Priority variant: ...
- A/B test hypothesis: ...
- Assets needed: ...
```

Length: One brief section per segment; total output typically 400–700 words. Use tables only if comparing more than 3 formats simultaneously.
