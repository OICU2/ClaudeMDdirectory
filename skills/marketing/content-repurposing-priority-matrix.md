# Content Repurposing Priority Matrix

This skill scores and ranks existing content assets by repurposing ROI — balancing adaptation effort against potential channel reach — so teams know exactly where to invest repurposing time first.

## Scoring Workflow

### Step 1: Inventory Content Assets
Ask the user to list their existing content assets or accept a provided list. For each asset, collect or estimate:
- Content type (blog post, video, webinar, podcast, whitepaper, etc.)
- Original performance metrics (views, shares, leads, engagement rate if available)
- Approximate age (fresh <6 months, mature 6–18 months, evergreen 18+ months)
- Topic category (educational, promotional, thought leadership, product-focused)

### Step 2: Score Each Asset on Four Dimensions
Rate each asset 1–5 on the following criteria:

**Reach Potential (1–5)**
- 1 = single niche channel, small audience ceiling
- 3 = 2–3 medium channels (e.g., LinkedIn + newsletter)
- 5 = 4+ high-volume channels (social, video, email, syndication, podcast)

**Repurposing Effort (1–5, inverted — lower effort = higher score)**
- 1 = requires full rewrite or new production (video → interactive tool)
- 3 = moderate editing and reformatting (webinar → blog series)
- 5 = low effort, format-friendly (long blog → LinkedIn carousel + Twitter thread + email)

**Content Longevity (1–5)**
- 1 = time-sensitive or news-dependent
- 3 = relevant for 6–12 months
- 5 = evergreen, no expiration

**Proven Performance (1–5)**
- 1 = no data or underperformed
- 3 = met average benchmarks
- 5 = top 20% performer in its category

### Step 3: Calculate Priority Score
For each asset:
`Priority Score = (Reach Potential × 2) + (Repurposing Effort × 2) + Content Longevity + Proven Performance`

Maximum score: 25. This double-weights reach and effort efficiency as the primary ROI drivers.

### Step 4: Map Repurposing Paths
For the top-ranked assets (score ≥ 18), specify exact repurposing actions:
- Identify which channels the asset can serve (LinkedIn, YouTube Shorts, email newsletter, podcast, SlideShare, Twitter/X threads, TikTok, etc.)
- Assign a format for each channel (carousel, clip, summary, script excerpt, infographic)
- Flag any assets that need a subject matter expert review before redistribution

### Step 5: Flag Blockers
Note any asset with a high score but a known blocker:
- Outdated statistics requiring updates
- Brand voice changes since original publication
- Legal/compliance review needed
- Rights restrictions on repurposed media

## Output Format

Produce a structured priority matrix with the following sections:

**1. Ranked Asset Table**
A markdown table with columns: Asset Name | Type | Reach (1–5) | Effort (1–5) | Longevity (1–5) | Performance (1–5) | Priority Score | Tier

Tier classification:
- **Tier 1 (Act Now):** Score 20–25
- **Tier 2 (Schedule Soon):** Score 14–19
- **Tier 3 (Low Priority):** Score ≤ 13

**2. Top 3 Repurposing Action Plans**
For the three highest-scored assets, provide:
- Asset name and current format
- Recommended repurposing formats (bulleted list)
- Target channels for each format
- Estimated effort level
