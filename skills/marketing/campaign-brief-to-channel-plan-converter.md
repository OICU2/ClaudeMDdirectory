# Campaign Brief to Channel Plan Converter

This skill converts a high-level marketing campaign brief into a structured, channel-by-channel execution plan with specific format guidance, timing, and messaging priorities for each platform.

## Conversion Workflow

### Step 1: Extract Brief Essentials
Pull these elements from the brief before planning:
- **Campaign objective** (awareness, consideration, conversion, retention)
- **Target audience** (demographics, psychographics, behavioral signals)
- **Core message / value proposition**
- **Budget tier** (if stated or implied: small <$10k, mid $10k–$100k, large $100k+)
- **Timeline / campaign window**
- **Brand constraints** (tone, visual rules, restricted platforms)

If any of these are missing, note the gap and state the assumption used to proceed.

### Step 2: Map Objective to Channel Mix
Apply this objective-to-channel logic:

| Objective | Priority Channels |
|---|---|
| Awareness | Paid social (Meta/TikTok/YouTube), Display, Programmatic, OOH |
| Consideration | Search (Google/Bing), Content/SEO, Email nurture, Retargeting |
| Conversion | Search, Retargeting, Affiliate, Direct email, Landing pages |
| Retention | Email, SMS, Loyalty push, Community (Discord/Slack/Reddit) |

Select 3–6 channels appropriate to objective, audience, and budget. Exclude channels that are misaligned with audience demographics or budget constraints.

### Step 3: Build Per-Channel Execution Detail
For each selected channel, define:

1. **Role in campaign** — what job this channel does (reach, educate, convert, retain)
2. **Recommended formats** — specific ad units, content types, or post formats (e.g., "15s vertical video reels," "responsive search ads," "plain-text nurture email sequence")
3. **Messaging priority** — which part of the core message leads on this channel and why
4. **Timing / sequencing** — when this channel activates relative to campaign start, and at what frequency or flight schedule
5. **Key metrics** — 2–3 KPIs to measure success on this channel specifically
6. **Creative notes** — tone, length, visual direction, CTA language appropriate to the platform

### Step 4: Define Campaign Sequencing
Map channels onto a campaign timeline with three phases:
- **Launch phase** (Days 1–7): Channels that build reach and awareness first
- **Engagement phase** (Weeks 2–4): Channels that deepen consideration or drive action
- **Conversion / closing phase** (Final weeks): Channels focused on bottom-funnel action and retention handoff

Show which channels overlap, which are sequential, and which run the full duration.

### Step 5: Flag Risks and Dependencies
Note:
- Channels requiring long lead times (SEO, influencer partnerships, OOH production)
- Budget allocation conflicts if budget is constrained
- Audience overlap risks between channels
- Any missing assets needed to execute (video, landing page, offer)

## Output Format

Produce a structured plan with the following sections in order:

---

**Campaign Summary**
One paragraph restating the campaign objective, audience, core message, and timeline as understood.

**Assumptions Made**
Bullet list of any gaps in the brief and the assumption used to fill each.

**Recommended Channel Mix**
A table listing each selected channel, its campaign role, and budget priority (High / Medium / Low).

**Channel Execution Plans**
One dedicated subsection per channel, each containing:
- Channel name as heading
- Role, Formats, Messaging Priority, Timing, Key Metrics, Creative Notes as labeled fields

**Campaign Sequencing Timeline**
A text
