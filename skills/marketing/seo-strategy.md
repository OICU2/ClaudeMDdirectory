# SEO Strategy

This skill builds actionable SEO strategies—from keyword research to technical audits to content plans—to increase a website's organic search visibility and traffic.

## SEO Strategy Workflow

### 1. Gather Context
Ask for and collect:
- Website URL and niche/industry
- Target audience and geography
- Current monthly organic traffic (if known)
- Top 3 competitors
- Primary business goal (leads, sales, signups, brand awareness)
- Existing content assets or blog

### 2. Define SEO Goals and KPIs
Set specific targets:
- Traffic goal (e.g., 10,000 organic visits/month in 6 months)
- Keyword ranking targets (e.g., top 5 for 3 head terms)
- Conversion metric (e.g., 2% organic-to-lead rate)
- Timeline (30/60/90-day milestones)

### 3. Keyword Research and Prioritization
Build a keyword matrix:
- **Head keywords**: High volume (1K–100K/month), high competition, long-term targets
- **Long-tail keywords**: Lower volume (100–1K/month), low competition, quick wins
- **Informational**: "how to," "what is," "best way to" — drives blog traffic
- **Commercial/transactional**: "buy," "best X for Y," "X pricing" — drives conversions
- For each keyword capture: search volume, keyword difficulty (0–100), search intent, current ranking (if any)

Prioritize by: low difficulty + high relevance + commercial intent first.

### 4. Competitive Gap Analysis
Compare the target site against 2–3 competitors:
- Identify keywords competitors rank for that the target site does not
- Note competitor content formats that rank (listicles, guides, tools, videos)
- Find backlink sources competitors use
- Spot content depth differences (word count, media, structured data)

### 5. Technical SEO Checklist
Flag and recommend fixes for:
- Page speed (Core Web Vitals — LCP < 2.5s, CLS < 0.1, FID < 100ms)
- Mobile-friendliness (responsive design)
- Crawlability (robots.txt, XML sitemap, no broken links)
- HTTPS and canonical tags
- Title tags and meta descriptions (unique, keyword-optimized, within character limits)
- Structured data / schema markup (FAQ, Article, Product where applicable)
- URL structure (short, descriptive, keyword-included)
- Internal linking (orphan pages, link equity distribution)

### 6. Content Strategy
Create a 90-day content plan:
- **Pillar pages**: 1–2 long-form (2,000–4,000 word) authoritative pages targeting head keywords
- **Cluster content**: 6–10 supporting articles (800–1,500 words) targeting long-tail variations, linked back to pillar pages
- **Content calendar**: Assign publish dates, target keyword, word count, and content type per piece
- Optimize existing pages before creating new ones if the site has existing content

Content optimization rules for each page:
- Primary keyword in H1, first 100 words, and URL
- Secondary keywords in H2/H3 subheadings
- Include FAQ section using People Also Ask questions
- Add internal links to 3+ related pages
- Include at least one original image, chart, or table

### 7. Link Building Plan
Recommend 3 tactics appropriate to the site's authority:
- **Low DA (0–30)**: Guest posting on niche blogs, directory submissions, HARO (Help a Reporter Out)
- **Mid DA (30–50)**: Broken link building, resource page outreach, original data/research publication
- **High DA (50+)**: Digital PR campaigns, partnership content, tool/calculator creation for passive links

Target: acquire 5–10 quality backlinks per month minimum.
