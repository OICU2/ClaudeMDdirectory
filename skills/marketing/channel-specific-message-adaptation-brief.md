# Channel-Specific Message Adaptation Brief

This skill takes a single core campaign message and produces ready-to-use creative briefs for paid social, email, organic search, and display, each with channel-appropriate format specs, tone guidance, and copy direction.

## Workflow

1. **Extract the core message** — Identify the campaign's single most important idea: the value proposition, emotional hook, or call to action. If unclear, ask the user to state it in one sentence before proceeding.

2. **Identify audience and objective** — Confirm the target audience segment and the primary campaign goal (awareness, consideration, conversion, retention). These shape tone and CTA intensity across all channels.

3. **Adapt for Paid Social (Meta/Instagram/TikTok/LinkedIn)**
   - Lead with the hook in the first 3 seconds or first line of copy
   - Specify primary format: static image, carousel, short-form video (under 30s), or story/reel
   - Tone: conversational, direct, platform-native (casual for Meta/TikTok, professional for LinkedIn)
   - Include: headline (under 40 characters), primary text (under 125 characters for feed), CTA button label
   - Flag any platform-specific restrictions (e.g., no before/after imagery on Meta)

4. **Adapt for Email**
   - Subject line (40–60 characters) and preview text (85–100 characters)
   - Email structure: hero statement → supporting proof point → single CTA
   - Tone: warmer, more conversational than paid; personalization tokens where relevant
   - Specify above-the-fold content, recommended body length (short: under 200 words for promotional; longer for nurture)
   - Include CTA button copy and placement (above fold, mid-email, or both)

5. **Adapt for Organic Search (SEO content / blog / landing page)**
   - Target keyword phrase and search intent (informational, navigational, transactional)
   - Content format: blog post, landing page, FAQ section, or pillar page
   - Tone: authoritative, helpful, clear — optimized for readability and E-E-A-T signals
   - Include: H1 suggestion, meta title (under 60 characters), meta description (under 160 characters), and 2–3 subheading ideas
   - Note recommended content length and internal linking opportunities

6. **Adapt for Display Advertising**
   - Core message compressed to 6-word headline maximum
   - Specify standard IAB sizes to produce: 300×250, 728×90, 160×600, 300×600
   - Tone: punchy, visual-first — copy supports the image, not the reverse
   - Include: headline, subheadline (optional, under 25 characters), CTA label (2–4 words)
   - Flag animation guidance: static vs. animated (3-frame max for standard display)

7. **Flag inconsistencies** — If the core message cannot work on a specific channel (e.g., a complex B2B value prop on TikTok), note the tension and suggest a modified angle rather than forcing a bad fit.

## Output Format

Produce one clearly labeled section per channel. Each section contains:

**Channel: [Name]**
- **Format:** [Specific ad unit or content type]
- **Tone:** [2–3 adjectives]
- **Headline / Subject Line:** [Final copy]
- **Body / Primary Text:** [Final copy or copy direction]
- **CTA:** [Exact label]
- **Notes:** [Any platform restrictions, personalization flags, or creative direction for the designer/developer]

Deliver all four channels in a single response. Use a horizontal rule between channels for readability. Total output should be scannable and ready to hand
