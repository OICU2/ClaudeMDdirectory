# Brand Guidelines Creation

This skill produces a comprehensive brand guidelines document by systematically capturing visual, tonal, and messaging standards that teams can reference to maintain brand consistency across all outputs.

## Workflow

1. **Gather Brand Inputs**
   - Ask for existing brand assets: logo files, color codes, fonts, sample copy, or previous style guides
   - If none exist, prompt for: brand mission, target audience, 3–5 brand personality adjectives, and competitor brands to differentiate from
   - Identify the primary use cases: digital, print, social, internal, or all

2. **Define Brand Identity Foundation**
   - Write a 2–3 sentence brand mission statement
   - Define brand personality using 4–6 attributes (e.g., bold, approachable, authoritative)
   - Identify target audience with demographics and psychographics
   - State key brand differentiators in 1–2 sentences

3. **Document Visual Identity**
   - **Logo Usage**: Specify primary, secondary, and icon-only versions; define clear space rules, minimum sizes, and prohibited uses (stretching, recoloring, drop shadows)
   - **Color Palette**: List primary and secondary colors with HEX, RGB, CMYK, and Pantone values; specify usage ratios (e.g., 60% primary, 30% secondary, 10% accent)
   - **Typography**: Name primary and secondary typefaces; specify weights, sizes, and hierarchy rules for headings, body, and captions; include web-safe fallbacks
   - **Imagery Style**: Describe photography tone (e.g., candid vs. staged), subject matter guidelines, and any illustration or icon style rules
   - **Spacing and Layout**: Define grid preferences, padding rules, and whitespace principles

4. **Define Brand Voice and Tone**
   - Write 3–4 core voice attributes with a one-line description each (e.g., "Direct — We say what we mean without jargon")
   - Create a "We are / We are not" contrast table (e.g., "Confident, not arrogant")
   - Specify tone variations by channel: social media, email, website, customer support
   - List grammar and punctuation preferences: Oxford comma, sentence case vs. title case, em dash usage

5. **Document Messaging Framework**
   - Write a single tagline or brand promise
   - Create a messaging hierarchy: elevator pitch (1 sentence), short description (2–3 sentences), full description (1 paragraph)
   - List 3–5 core value propositions with supporting proof points
   - Define key messages for each primary audience segment if multiple exist

6. **Compile Usage Examples and Anti-Examples**
   - Provide 2–3 on-brand copy examples for key channels
   - Provide 2–3 off-brand examples with explanations of what to avoid
   - Include visual do/don't references for logo and color misuse

7. **Add Governance Notes**
   - State who owns brand approval
   - List asset storage location or link placeholder
   - Define review cadence (e.g., annual refresh)

## Output Format

Produce a single, structured Markdown document with the following sections in order:

```
# [Brand Name] Brand Guidelines

## 1. Brand Identity
- Mission Statement
- Brand Personality
- Target Audience
- Key Differentiators

## 2. Visual Identity
- Logo Usage
- Color Palette (table with HEX/RGB/CMYK)
- Typography (table with typeface, weight, use case)
- Imagery Style
- Spacing and Layout

## 3. Voice and Tone
- Core Voice Attributes
- We Are / We Are Not Table
- Tone by Channel
- Grammar and Style Rules

## 4. Messaging Framework
- Tagline
- Brand Descriptions (short, medium, long)
- Core Value Propositions
- Audience-Specific Messages
