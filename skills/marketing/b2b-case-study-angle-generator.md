# B2B Case Study Angle Generator

This skill takes a single customer success story and extracts 6–10 distinct narrative angles, each optimized for a different buyer persona, funnel stage, or distribution channel.

## Angle Extraction Workflow

**Step 1 — Gather the raw story**
Ask for the source material if not provided: customer name/industry, the problem they faced, what solution they implemented, measurable outcomes achieved, and any direct quotes. If only partial information is available, proceed with what exists and note gaps.

**Step 2 — Identify the core facts**
Extract and list these elements explicitly before generating angles:
- Primary metric (e.g., "reduced churn by 34%")
- Secondary metrics or softer wins (e.g., "team saved 10 hours/week")
- Industry and company size
- Specific features or capabilities used
- Timeline from implementation to result
- Champion persona (who drove the decision)
- Economic buyer persona (who signed off)

**Step 3 — Generate angles across these dimensions**

For each angle, consider:

- **Persona angles**: Write one version for the technical buyer (features, integration, security), one for the economic buyer (ROI, payback period, risk reduction), one for the end user (workflow, ease of use, day-to-day impact)
- **Problem-first angles**: Lead with the pain point instead of the solution — useful for awareness-stage content and cold outreach
- **Metric-first angles**: Lead with the most impressive number — useful for paid ads, email subject lines, and executive summaries
- **Industry/vertical angles**: Reframe the story using language, regulations, or benchmarks specific to the customer's industry — useful for vertical campaigns
- **Competitive displacement angle**: If the customer switched from a competitor or manual process, frame the story around the "before vs. after" contrast
- **Implementation speed angle**: If time-to-value was fast, build an angle around low risk and quick wins — useful for late-stage sales objections
- **Expansion/scale angle**: If the customer grew usage over time, frame the story around long-term partnership and scalability
- **Executive/board angle**: Strip to a single outcome sentence plus one risk-mitigation point — useful for slide decks and briefing documents

**Step 4 — Map each angle to a use case**
For every angle generated, specify: recommended channel (e.g., email, LinkedIn, sales deck, landing page), funnel stage (awareness / consideration / decision), and target persona.

**Step 5 — Flag missing information**
Note which angles would be stronger with additional data (e.g., "the competitive displacement angle needs confirmation that they were previously using X").

## Output Format

Produce a structured document with the following layout:

---
**Source Story Summary** (3–5 bullet points covering the key facts extracted in Step 2)

---

For each angle, use this repeating block:

**Angle [#]: [Short angle name, e.g., "CFO ROI Angle"]**
- **Core message:** One sentence stating the central claim
- **Lead hook:** The opening line or headline (ready to use as-is)
- **Supporting points:** 3 bullet points that substantiate the claim
- **Best used for:** Channel + funnel stage + target persona
- **What's needed to strengthen this:** Any missing data or quotes that would improve it

---

Deliver 6–10 angles minimum. Keep each block tight — total output should be scannable in under 5 minutes. End with a **Quick-Pick Table** that maps all angles to personas and channels in a single reference grid.
