# Cross-Channel Message Sequencing Plan

This skill builds a coordinated, channel-by-channel messaging timeline that ensures prospects receive a logical narrative progression instead of redundant or contradictory content across email, paid ads, and organic touchpoints.

## Sequencing Workflow

### 1. Extract Campaign Inputs
Before building the sequence, collect or infer:
- **Goal**: Awareness, lead gen, nurture, conversion, retention
- **Audience segment**: Cold, warm, existing customer
- **Channels in play**: Email, paid search, paid social, organic social, SEO content, SMS, retargeting
- **Campaign window**: Days, weeks, or ongoing
- **Core offer or message**: What is ultimately being sold or communicated

If any of these are missing, ask before proceeding.

### 2. Define the Narrative Arc
Map the message progression to a 3-stage funnel:
- **Stage 1 — Problem/Awareness**: Surface the pain point or opportunity. No selling.
- **Stage 2 — Solution/Consideration**: Introduce the offering as a credible answer. Build trust.
- **Stage 3 — Decision/Conversion**: Present the specific offer, CTA, urgency, or proof.

Each channel should carry the stage-appropriate message, not the same message.

### 3. Assign Channels to Stages and Timing

Apply these channel-role defaults (adjust based on actual channel mix):

| Stage | Best Channels | Message Role |
|---|---|---|
| Awareness | Organic social, paid social (cold), SEO blog | Educate, provoke curiosity |
| Consideration | Email (nurture), retargeting ads, organic social | Build trust, address objections |
| Conversion | Email (offer), retargeting (urgency), paid search | CTA, proof, scarcity |

- Assign each channel a **primary stage** so no two channels compete with the same message simultaneously.
- Stagger timing so channels reinforce each other sequentially, not simultaneously.

### 4. Build the Day-by-Day or Week-by-Week Sequence

For each time unit, specify:
- Which channel fires
- What message or theme is delivered
- What the previous touchpoint was (so this one builds on it)
- What action the prospect is expected to take (or not take — sometimes the goal is impression-only)

Enforce these sequencing rules:
- **No channel should repeat the exact same offer before the prospect has moved to the next stage.**
- **Email should not lead with conversion messaging if the prospect has only seen top-of-funnel paid ads.**
- **Retargeting ads should reference or mirror the last email theme for continuity.**
- **Organic content should be published before paid amplification of the same topic.**

### 5. Identify Conflict Points and Gaps

Review the draft sequence and flag:
- **Conflicts**: Two channels delivering contradictory CTAs at the same time (e.g., email says "Book a call" while ad says "Download the free guide")
- **Redundancy**: Same message across three channels in the same week with no variation
- **Gaps**: Stage jumps where prospect hasn't been warmed before a conversion ask
- **Orphaned channels**: A channel that runs independently with no logical connection to the others

Resolve each flagged issue before finalizing.

### 6. Define Suppression and Progression Logic

Specify trigger conditions that move a prospect through the sequence or remove them:
- If prospect clicks email CTA → suppress from awareness-stage ads, move to consideration emails
- If prospect converts → remove from all nurture sequences immediately
- If prospect ignores 3+ emails → shift emphasis to paid retargeting with a different angle
- If prospect engages with organic content → accelerate email cadence
