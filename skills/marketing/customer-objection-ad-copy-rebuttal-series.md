# Customer Objection Ad Copy Rebuttal Series

This skill converts a list of customer objections into a structured series of ad creatives that reframe hesitations as reasons to buy, deployable across paid social, search, and display channels.

## Objection-to-Ad Conversion Workflow

**Step 1: Intake and Categorize Objections**
- Ask the user to provide 3–10 objections verbatim from sales calls or CRM notes
- If none are provided, prompt: "What are the top reasons prospects say no or delay?"
- Categorize each objection into one of five types: Price, Trust, Timing, Complexity, or Competitor

**Step 2: Identify the Hidden Fear Behind Each Objection**
- For each objection, extract the underlying fear or belief driving it
- Example: "It's too expensive" → Fear: "I'll pay and not see results"
- This fear becomes the emotional core of the ad

**Step 3: Develop the Rebuttal Angle**
- For each objection, choose one of three rebuttal strategies:
  - **Flip**: Turn the objection into a proof point ("Too expensive? Our customers recoup costs in 60 days")
  - **Validate + Redirect**: Acknowledge the concern, then redirect ("Fair concern — here's what changed")
  - **Social Proof Shield**: Use a customer story that mirrors the objection and resolves it

**Step 4: Write Ad Creatives Per Objection**
- Write one complete ad set per objection using the chosen rebuttal strategy
- Each ad set includes: headline, primary text, call-to-action, and suggested visual direction

**Step 5: Sequence the Series**
- Order ads from highest-friction to lowest-friction objections
- Label each ad by funnel stage: Awareness, Consideration, or Decision
- Note which objections are best addressed via retargeting vs. cold audiences

## Output Format

Produce one clearly labeled section per objection with the following structure:

---
**Objection #[N]: "[Verbatim objection]"**
- Objection Type: [Price / Trust / Timing / Complexity / Competitor]
- Hidden Fear: [1 sentence]
- Rebuttal Strategy: [Flip / Validate + Redirect / Social Proof Shield]
- Funnel Stage: [Awareness / Consideration / Decision]
- Audience: [Cold / Retargeting]

**Ad Creative:**
- Headline (≤10 words): [Headline]
- Primary Text (2–4 sentences): [Body copy that opens with the objection signal and closes with proof or urgency]
- Call-to-Action: [Specific CTA button text]
- Visual Direction: [1-sentence description of image or video concept]
---

Deliver all objections as a complete numbered series. End with a brief sequencing note (3–5 sentences) recommending the campaign rollout order and which objections to A/B test against each other.
