# Customer Advocacy Program Design

This skill builds a complete, actionable advocacy program framework that moves satisfied customers through a structured pipeline from identification to active public championing across reviews, referrals, and co-marketing.

## Program Design Workflow

### 1. Gather Context First
Ask for (or infer from conversation):
- Product/service type and target customer profile
- Current customer base size and NPS/satisfaction signal availability
- Existing touchpoints (CSM team, support, community, email)
- Goals: reviews, referrals, case studies, co-marketing, or all
- Budget range: zero-budget, low-budget (<$10K), or full program

### 2. Define the Advocacy Tiers
Build a 3-tier structure:

**Tier 1 — Engaged Fans** (low-lift, high volume)
- Actions: G2/Capterra reviews, social shares, survey participation
- Recruitment trigger: NPS 9–10, milestone achievement, renewal
- Incentive: Recognition, early access, swag

**Tier 2 — Active Champions** (medium-lift, medium volume)
- Actions: Referrals, testimonials, community moderation, beta programs
- Recruitment trigger: Repeat engagement, product power-user behavior
- Incentive: Referral credits, co-branding, dedicated CSM support

**Tier 3 — Strategic Partners** (high-lift, low volume)
- Actions: Case studies, speaking slots, co-authored content, advisory board
- Recruitment trigger: Logo value, expansion revenue, strong narrative fit
- Incentive: Revenue share, executive access, joint PR, named features

### 3. Build the Identification System
Define criteria for flagging advocacy candidates:
- Primary signals: NPS ≥9, CSAT ≥4.5, renewal without negotiation, expansion purchase
- Behavioral signals: Feature adoption depth, community posts, unprompted referrals, social mentions
- Relationship signals: Positive CSM notes, support tickets resolved happily, executive engagement

Specify the data source for each signal (CRM, product analytics, support tool, NPS platform).

### 4. Design the Recruitment Sequence
Map a 3-step outreach flow per tier:
- **Step 1:** Warm ask from CSM or champion team (personalized, reference specific win)
- **Step 2:** Value proposition email with clear ask + benefit (no ambiguity on effort required)
- **Step 3:** Follow-up with social proof (existing advocate quote or program outcome stat)

Include timing: first ask at success milestone, not at renewal conversation.

### 5. Define the Activation Playbook
For each advocacy action, provide:
- The specific ask (exact format: "We'd love a 3-sentence review on G2 focused on [outcome]")
- Enablement assets needed (review template, referral link, case study intake form)
- Timeline and deadline
- Who owns the relationship (CSM, community manager, marketing)

### 6. Set the Measurement Framework
Define KPIs per program goal:
- **Reviews:** Volume per quarter, star rating average, review site coverage
- **Referrals:** Referral-sourced pipeline, conversion rate, revenue attributed
- **Case Studies:** Number published per quarter, pipeline influenced
- **Co-marketing:** Impressions, lead volume, content pieces produced
- **Health metric:** Advocate churn rate (advocates who go silent or opt out)

### 7. Address Retention and Renewal of Advocates
- Set a quarterly touchpoint cadence for Tier 2 and Tier 3
- Rotate asks so no advocate is over-leveraged
- Define an off-ramp process for churned customers or advocates who disengage
- Build in appreciation moments: anniversary recognition, milestone shoutouts
