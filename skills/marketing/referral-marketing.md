# Referral Marketing Skill

This skill helps design end-to-end referral programs including incentive structures, messaging, mechanics, and launch strategy to systematically convert satisfied customers into a growth channel.

## Referral Program Design Process

### 1. Define Program Goals
- Clarify primary objective: new customer acquisition, revenue growth, or market expansion
- Establish target referral conversion rate and volume benchmarks
- Identify the customer segment most likely to refer (power users, recent purchasers, high-NPS respondents)

### 2. Choose Incentive Structure
Select the reward model based on business type and margin:

| Model | Best For | Example |
|---|---|---|
| Double-sided reward | SaaS, subscriptions | $20 credit for referrer + $20 for new user |
| One-sided (referrer only) | High-margin products | $50 cash per successful referral |
| Tiered rewards | High-volume programs | 1 ref = $10, 5 refs = $75, 10 refs = $200 |
| Non-cash perks | Brand-driven products | Early access, exclusive status, free upgrades |
| Revenue share | Affiliate-style programs | 15% commission on referred revenue |

Rules for incentive design:
- Reward must feel valuable relative to the ask — friction and reward must match
- Cash or equivalent (credits, gift cards) outperforms branded swag
- New customer discount lowers their barrier to convert; referrer reward drives action
- Reward only on qualified conversion (purchase, activation, subscription start), not signup

### 3. Define Referral Mechanics
- **Trigger point:** Identify when to ask (post-purchase, after first value moment, after positive support interaction)
- **Sharing method:** Unique link, promo code, or email invite tool
- **Tracking:** Cookie-based attribution for links, code redemption for promo codes
- **Expiry:** Set link/code validity window (30–90 days recommended)
- **Fraud prevention:** Limit self-referrals, flag same-device conversions, cap rewards per account

### 4. Write Referral Messaging
Structure the ask with three elements:
1. **Social proof hook** — "Customers like you save an average of $X"
2. **Clear offer** — exactly what both parties get, in plain language
3. **Single CTA** — one button, one link, no decision paralysis

Channel-specific copy:
- **In-app prompt:** 1–2 sentences max, dismissible, shown after value moment
- **Email:** Subject line leads with the reward, body explains the mechanic in 3 bullet points
- **Post-purchase page:** Banner or modal with pre-filled share message for social/SMS

### 5. Set Up the Program Flow
Map the full user journey:
1. Existing customer sees referral prompt → clicks CTA
2. Receives unique share link or code
3. Shares via email, SMS, or social
4. New user clicks link → lands on co-branded or offer-highlighted page
5. New user converts (purchases, subscribes, activates)
6. Both parties receive reward automatically or within defined SLA
7. Referrer receives confirmation notification

### 6. Launch and Optimize
- Seed the program by emailing your highest-NPS customers first
- A/B test incentive amounts (e.g., $10 vs. $25 vs. free month)
- Track: referral rate, share rate, click-to-conversion rate, cost per referred acquisition
- Review monthly; kill underperforming channels, double down on highest-converting share methods
