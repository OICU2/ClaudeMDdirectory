# Podcast Guest Pitch Sequence Builder

This skill builds a complete, personalized podcast guest outreach system — from host research to follow-up emails — so users can land guest spots without sounding generic or spammy.

## Workflow

### Step 1: Gather User Context
Ask for the following if not provided:
- Their name, title, and area of expertise
- Their core talking points or stories (2–3 topics they want to discuss)
- Target podcast name and host name (or audience niche if no specific show)
- Any mutual connections, shared experiences, or prior mentions of the host's work
- Their credibility signals (books, stats, prior press, notable clients)

### Step 2: Build the Host Research Brief
Produce a compact research brief the user can use before sending anything. Include:
- **What to research**: recent episodes, host's stated content gaps, guest formats they prefer, host's personal projects or social posts
- **Personalization hooks**: 2–3 specific angles to reference in outreach (e.g., an episode they loved, a gap they can fill, a topic the host has teased wanting to cover)
- **Red flags to avoid**: topics the host has already covered extensively, guest types they've complained about

### Step 3: Write the Initial Pitch Email
Structure:
1. **Subject line** (3 options): specific, curiosity-driven, under 8 words each
2. **Opening line**: reference something real and specific about the host or show (not generic praise)
3. **The hook**: one sentence on why this guest + this show = perfect fit right now
4. **The pitch**: 2–3 proposed episode angles with working titles
5. **Credibility proof**: 1–2 sentences max, results-focused
6. **Clear CTA**: one ask, frictionless (e.g., "Would a 15-minute intro call work?")
7. **Total length**: 150–200 words

### Step 4: Write the Follow-Up Cadence
Produce 3 follow-up messages:

- **Follow-up 1 (Day 5)**: Bump the thread, add one new value-add (a recent stat, a timely news hook, or a short story that fits their audience)
- **Follow-up 2 (Day 12)**: Reframe the pitch from a different angle; acknowledge they're busy; offer an alternative format (solo Q&A, panel, short segment)
- **Follow-up 3 (Day 21)**: Graceful final touch — leave the door open, no pressure, mention you'll share their show regardless

Each follow-up: 75–100 words.

### Step 5: Deliver a One-Sheet Template
Create a simple guest one-sheet outline with:
- Bio (2 versions: 50 words and 150 words)
- 5 ready-to-go episode topic titles with one-line descriptions
- 3–5 sample interview questions the host can use
- Social proof block (links, stats, or quotes placeholder)
- Contact and booking info placeholder

## Output Format

Deliver the full sequence as a single structured document with clearly labeled sections:

```
## HOST RESEARCH BRIEF
[bullet points]

## INITIAL PITCH EMAIL
Subject line options: ...
Email body: ...

## FOLLOW-UP SEQUENCE
Follow-Up 1 — Day 5: ...
Follow-Up 2 — Day 12: ...
Follow-Up 3 — Day 21: ...

## GUEST ONE-SHEET TEMPLATE
[structured sections]
```

- Use plain text for all emails (no HTML, no heavy formatting inside the email bodies)
- Flag any section where the user needs to insert personalized research with `[CUSTOMIZE: ...]`
- Do not write generic filler — if you lack specific details, ask before guessing
