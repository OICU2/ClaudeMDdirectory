# Organic Social Content Calendar

Generates a full 7-day, platform-specific social media content calendar with post themes, formats, copy angles, and optimal publishing times based on the brand, audience, and goals provided.

## Calendar Building Workflow

1. **Gather inputs before building** — Ask for: target platforms (Instagram, LinkedIn, TikTok, X, Facebook, Pinterest), brand/business type, target audience, primary goal (awareness, engagement, leads, sales), content pillars or topics if known, and posting frequency preference. If the user provides these upfront, skip asking.

2. **Define content pillars** — Identify 3–5 repeating content themes based on the brand and goal. Examples: educational tips, behind-the-scenes, social proof, promotional, trending/timely, community-driven. Every post must map to a pillar.

3. **Match format to platform** — Assign formats appropriate to each platform:
   - Instagram: Reels, carousels, static posts, Stories
   - LinkedIn: long-form text posts, document carousels, short insights, polls
   - TikTok: short-form video concepts (15–60s), trending audio hooks
   - X (Twitter): threads, single punchy takes, question prompts
   - Facebook: link posts, video, community questions
   - Pinterest: idea pins, static infographics, step-by-step visuals

4. **Write copy angles** — For each post, provide a specific angle or hook, not a vague topic. Instead of "post about your product," write: "Hook: 'We almost didn't launch this — here's why.' Angle: vulnerability + product origin story."

5. **Assign publishing times** — Use platform-specific engagement windows:
   - Instagram: Tue–Fri, 9–11am or 6–8pm local time
   - LinkedIn: Tue–Thu, 7–9am or 12–1pm
   - TikTok: Mon/Wed/Fri, 7–9am or 7–11pm
   - X: Weekdays, 8–10am or 5–7pm
   - Facebook: Wed–Fri, 1–4pm

6. **Balance the week** — Vary pillar types across the week. Never stack two promotional posts back to back. Open Monday with high-value or educational content. End Friday with engaging, lightweight content (polls, questions, humor).

7. **Flag content that needs assets** — Mark posts that require video shooting, graphic design, or photography so the user knows what to prepare in advance.

## Output Format

Produce a structured 7-day calendar table followed by post briefs.

**Section 1 — Weekly Calendar Table**
One table per platform requested. Columns: Day | Pillar | Format | Topic/Hook | Post Time | Asset Needed (Y/N)

**Section 2 — Post Briefs**
For each day, write a brief block:
- **Day + Platform**
- **Pillar:** [name]
- **Format:** [e.g., Carousel, Reel, Text Post]
- **Hook/Opening Line:** [exact first line or hook]
- **Copy Angle:** [2–3 sentences describing the narrative, tone, and CTA]
- **Hashtag Strategy:** [3–5 relevant hashtags or hashtag categories]
- **Asset Notes:** [what needs to be created or sourced]

**Length:** Full 7-day calendar = 7 post briefs per platform. If multiple platforms requested, group by platform. Keep each brief scannable — no paragraph walls.

**Tone:** Match the brand voice if specified. Default to clear, direct, and actionable.
