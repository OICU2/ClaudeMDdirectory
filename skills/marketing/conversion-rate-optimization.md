# Conversion Rate Optimization

This skill diagnoses why visitors aren't converting and produces a prioritized list of specific, testable improvements across copy, design, UX, and trust signals.

## CRO Audit Workflow

### 1. Clarify the Conversion Goal
- Identify the primary action: purchase, sign-up, demo request, download, click, etc.
- Confirm the page type: landing page, homepage, product page, checkout, email, ad, etc.
- Ask for current conversion rate and traffic source if not provided.

### 2. Diagnose the Core Problem
Evaluate the page or funnel against these failure categories:

**Clarity failures**
- Headline does not communicate the value proposition within 5 seconds
- CTA button copy is generic ("Submit," "Click Here") instead of outcome-focused
- Benefit statements lead with features, not user outcomes

**Trust failures**
- Missing or weak social proof (testimonials, logos, review counts, case studies)
- No risk-reversal elements (money-back guarantee, free trial, no credit card required)
- Unprofessional design, broken elements, or slow load time signals

**Friction failures**
- Too many form fields for the value being offered
- Checkout requires account creation before purchase
- Too many steps between intent and completion
- Distracting navigation or competing CTAs pulling attention away

**Message-market match failures**
- Page copy doesn't reflect the language of the ad or referral source
- Value proposition doesn't match audience awareness level (cold vs. warm traffic)
- Offer doesn't match where the visitor is in the buying journey

**Urgency and motivation failures**
- No reason to act now vs. later
- Benefits are abstract rather than concrete and specific
- Lack of scarcity, deadlines, or loss-framing where appropriate

### 3. Prioritize by Impact
Rank issues using PIE framework:
- **P**otential: How much improvement is possible?
- **I**mportance: How much traffic or revenue does this affect?
- **E**ase: How quickly can it be implemented?

Score each issue 1–10 on each dimension. Highest PIE score = fix first.

### 4. Write Specific Recommendations
For each issue:
- State the problem precisely
- Explain why it hurts conversions (psychological or behavioral reason)
- Provide the exact fix or rewrite
- Suggest an A/B test hypothesis if applicable

### 5. Propose Quick Wins vs. Strategic Changes
- **Quick wins (0–2 days):** Headline rewrites, CTA copy changes, adding a testimonial, removing form fields
- **Medium effort (1–2 weeks):** Page restructure, new social proof section, trust badge placement, offer adjustment
- **Strategic (1+ month):** Full page redesign, funnel restructure, new offer creation, user research
