# Copywriting Strategy Skill

This skill develops full-funnel copy strategies and produces channel-specific copy that aligns with brand voice, audience psychology, and conversion goals.

## Copy Strategy Workflow

1. **Define the audience** — Identify the target persona: demographics, pain points, desires, objections, and level of awareness (problem-aware, solution-aware, product-aware).

2. **Clarify the goal** — Determine the single conversion action: sign-up, purchase, click, reply, download, or call. Every piece of copy serves one goal.

3. **Establish brand voice** — Extract or define tone attributes (e.g., authoritative, conversational, urgent, empathetic). Apply consistently across all deliverables.

4. **Map the channel** — Adapt copy length, format, and hook style to the channel:
   - **Landing pages**: headline + subhead + benefit bullets + CTA
   - **Email sequences**: subject line + preview text + body + CTA
   - **Paid ads**: hook + value prop + CTA within character limits
   - **Social posts**: pattern interrupt + value + engagement prompt
   - **SMS**: ultra-short, action-first, single CTA

5. **Apply proven copy frameworks** — Choose based on context:
   - **AIDA** (Attention → Interest → Desire → Action) for long-form
   - **PAS** (Problem → Agitate → Solution) for pain-driven audiences
   - **FAB** (Features → Advantages → Benefits) for product copy
   - **Before/After/Bridge** for transformation messaging

6. **Write the copy** — Lead with the strongest hook. Use active voice, specific numbers, and power words. Eliminate jargon unless audience-appropriate. Mirror the reader's internal language.

7. **Add conversion elements** — Include urgency or scarcity where authentic, social proof references, risk-reversal language, and a single, clear CTA.

8. **Review for friction** — Cut any sentence that doesn't advance the reader toward the goal. Flag weak CTAs, passive constructions, and vague claims.

## Output Format

Deliver a structured copy package containing:

- **Strategy Brief** (3–5 lines): audience, goal, voice, framework chosen
- **Copy Deliverables**: fully written copy for each requested channel, clearly labeled
  - Each piece includes headline/subject, body, and CTA
  - Character or word counts noted where channel limits apply
- **A/B Variants**: provide 2 headline or subject line variants per deliverable
- **Copy Notes** (optional): 1–3 sentences explaining key strategic choices or testing recommendations

Format all copy in code blocks or clearly separated sections for easy extraction and use.
