# Customer Testimonial Campaign

This skill builds an end-to-end testimonial campaign system — from identifying ideal candidates and crafting outreach, to formatting collected testimonials and deploying them across marketing channels.

## Campaign Workflow

### 1. Define Testimonial Goals
- Identify which product/service, audience segment, or use case needs coverage
- Determine deployment channels (website, ads, email, sales decks, social)
- Clarify what objections or buying decisions the testimonials should address
- Set a target: number of testimonials, format types (written, video, case study), and deadline

### 2. Build the Candidate List
- Pull from: recent NPS promoters (9–10), repeat buyers, customers who submitted positive support tickets, or referral sources
- Prioritize customers who represent the target buyer persona
- Note each candidate's industry, use case, and known results for personalization

### 3. Create Outreach Sequence
Write a 3-touch email sequence:
- **Email 1 (Ask):** Personal, specific, low-friction. Reference their result. Link to a short form or offer a 10-minute call.
- **Email 2 (Follow-up, Day 5):** Brief nudge. Offer to write a draft they can edit.
- **Email 3 (Final, Day 10):** Last ask. Offer alternative formats (phone call, 2-sentence reply, LinkedIn recommendation).

Include a testimonial intake form with questions:
1. What problem were you solving before using [product/service]?
2. What results have you seen since?
3. Who would you recommend this to and why?
4. Can we use your name, title, and photo?

### 4. Collect and Edit Testimonials
- Transcribe or copy raw responses exactly first
- Edit for clarity and length without changing meaning or voice — flag any edits made
- Get written approval on the final version before publishing
- Capture: full name, title, company, headshot, company logo (for B2B)

### 5. Format for Each Channel
| Channel | Format | Length |
|---|---|---|
| Homepage hero | Pull quote + photo + name/title | 1–2 sentences |
| Case study | Story arc: problem → solution → result | 300–600 words |
| Email signature block | Short quote + attribution | 1 sentence |
| Social post | Quote graphic + brief caption | 15–25 words |
| Sales deck slide | 2–3 quotes grouped by use case | 1–2 sentences each |
| Google/G2/Capterra | Unedited customer voice, directed to review site | 75–150 words |

### 6. Deploy and Track
- Schedule testimonials into editorial calendar across channels
- Tag by persona, use case, and objection addressed for easy retrieval
- Track which testimonials appear in high-converting assets (A/B test on landing pages)
- Set a quarterly cadence to refresh the pipeline

## Output Format

Deliver a campaign package containing:

1. **Campaign Brief** (bullet list): goals, target count, timeline, channels, buyer personas to cover
2. **Candidate List Template** (table): Name | Company | Use Case | Known Results | Outreach Status
3. **3-Email Outreach Sequence**: subject lines, full body copy, personalization placeholders in [brackets]
4. **Intake Form**: 4–6 questions formatted for Google Forms, Typeform, or email reply
5. **Formatted Testimonial Examples**: 2–3 sample testimonials written in each required channel format based on any provided customer data
6. **Deployment Checklist**: channel-by-channel list of where each testimonial type gets placed, with approval and tagging steps

Length: comprehensive enough to hand off to a marketing coordinator. Use headers, tables, and numbered lists throughout. No narrative padding.
