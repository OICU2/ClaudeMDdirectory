# Brand Voice Onboarding Guide Builder

This skill creates a complete, writer-ready brand voice guide with tone descriptions, real examples, do-and-don't pairs, and vocabulary references so new contributors can write on-brand from day one.

## Guide Construction Workflow

1. **Gather brand inputs** — Ask the user for: brand name, industry, target audience, 3–5 adjectives describing their voice, any existing sample copy (good and bad), and any words/phrases that are banned or overused.

2. **Define the voice in plain language** — Write a 2–3 sentence "Voice Snapshot" that describes the brand's personality as if it were a person. Example: "We write like a knowledgeable friend, not a textbook. Direct, warm, and never condescending."

3. **Build the tone spectrum** — Create a 4-quadrant tone reference showing where the brand sits on axes: Formal ↔ Casual and Serious ↔ Playful. Add a one-line rule for when to adjust tone (e.g., customer complaints = more formal, social posts = more casual).

4. **Write do-and-don't pairs** — Generate at least 8 side-by-side examples covering: headlines, CTAs, error messages, product descriptions, and social captions. Each pair must show the wrong version and the corrected on-brand version with a one-line explanation of why.

5. **Build the word choice reference** — Three columns: "Use These" (approved power words), "Avoid These" (off-brand or overused terms), and "Never Say These" (banned phrases, competitor names, or legally flagged terms).

6. **Add sentence-level style rules** — List 5–8 concrete grammar and formatting rules specific to the brand: sentence length preference, Oxford comma stance, how to handle numbers, capitalization quirks, emoji policy, and punctuation habits.

7. **Include a "When In Doubt" checklist** — A 5-question self-review writers run before submitting copy: Does it sound like us? Would our audience understand this immediately? Does it avoid jargon? Is it the shortest version that still works? Does it match the channel tone?

## Output Format

Produce a single, structured Markdown document with the following sections in order:

- **Brand Voice Snapshot** (2–3 sentences)
- **Who We're Writing For** (audience summary, 3–4 bullet points)
- **Tone Spectrum** (table or labeled quadrant description)
- **Do This / Not That** (minimum 8 pairs in a two-column table with explanation column)
- **Word Choice Reference** (three-column table: Use / Avoid / Never)
- **Style Rules** (numbered list, 5–8 rules)
- **Channel Tone Adjustments** (brief table: Email / Social / Web / Ads / Support)
- **When In Doubt Checklist** (5 yes/no questions)

Target length: 600–900 words. Use plain Markdown. No decorative headers or filler copy. Every section must contain information specific to the brand provided, not generic placeholder text.
