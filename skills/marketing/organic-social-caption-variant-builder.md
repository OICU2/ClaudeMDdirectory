# Organic Social Caption Variant Builder

This skill takes a single piece of source content and produces platform-optimized caption variants for each specified social channel, respecting the unique tone, character limits, hashtag conventions, and audience behavior of each platform.

## Workflow

1. **Extract core message**: Identify the central idea, offer, or story from the source content. Strip it to its essential value — what should the audience feel, know, or do?

2. **Clarify inputs if missing**: If the user hasn't provided source content, ask for it. If platforms aren't specified, default to LinkedIn, Instagram, X/Twitter, Facebook, and TikTok. If a call-to-action isn't clear, ask or infer from context.

3. **Apply platform rules before writing**:
   - **LinkedIn**: Professional tone, 1st-person narrative or insight-driven, 150–300 words, line breaks for scannability, 3–5 relevant hashtags at the end, no emojis or sparse use only
   - **Instagram**: Conversational and visual-forward, lead with a hook in the first line (visible before "more"), 100–200 words, 10–20 hashtags in first comment or end of caption, 2–5 emojis used purposefully
   - **X/Twitter**: Punchy, direct, opinionated or curious, under 280 characters for the primary tweet, optional thread continuation marked as [Thread 1/], [Thread 2/], 1–2 hashtags max
   - **Facebook**: Warm, community-oriented tone, 50–150 words, a direct question to prompt comments, 1–3 hashtags, emojis optional
   - **TikTok**: Casual, trend-aware, energetic, 100–150 words, hook in first sentence, 3–5 trending or niche hashtags, CTA encouraging saves or comments

4. **Write each variant independently**: Do not copy-paste across platforms. Each caption must feel native — different sentence structures, different hooks, different CTAs where appropriate.

5. **Check for brand voice consistency**: All variants should feel like they come from the same brand even though the tone shifts. Core message and key terms should remain consistent.

6. **Add a hashtag block where applicable**: For Instagram and TikTok, provide a dedicated hashtag set beneath the caption copy.

## Output Format

Produce a clearly labeled section for each platform in this structure:

---

**[Platform Name]**
[Caption copy]

*Hashtags (if applicable):* [hashtag list]

---

Repeat for each platform. End with a brief **"Remix Notes"** section (2–4 bullet points) flagging any platform where the message required significant reframing, suggesting A/B test angles, or noting where a visual or hook asset would strengthen performance.

Total output length will vary by number of platforms but typically runs 400–700 words across all variants.
