# Product Positioning Competitive Gap Analysis

This skill maps competitor positioning claims against unmet customer needs to surface white space opportunities a product can own, and activates whenever someone needs to find a defensible, differentiated market position.

## Competitive Gap Mapping Workflow

**Step 1: Extract Competitor Positioning Claims**
- Ask the user to list 3–8 direct and indirect competitors
- For each competitor, identify: primary value proposition, target audience, key messaging pillars (1–3 phrases), and claimed differentiators
- If the user lacks this data, prompt them to pull it from competitor homepages, G2/Capterra reviews, or LinkedIn ads

**Step 2: Build the Attribute Matrix**
- List every positioning dimension mentioned across all competitors (e.g., price, speed, ease of use, enterprise-grade, compliance, integrations, support quality)
- Create a grid: competitors as rows, attributes as columns
- Mark which attributes each competitor owns, claims weakly, or ignores entirely
- Highlight columns where fewer than two competitors make strong claims — these are candidate gaps

**Step 3: Map Customer Need Evidence**
- Ask the user for evidence of unmet customer needs: support tickets, sales call objections, review site complaints, churn reasons, or user research quotes
- Categorize each need against the attribute matrix
- Flag any customer need that appears in multiple sources but is absent or weak in competitor messaging — these are high-value gaps

**Step 4: Score and Rank Gap Opportunities**
- For each identified gap, score on three dimensions (1–5):
  - **Customer urgency**: How painfully unmet is this need?
  - **Competitive whiteness**: How few competitors own this space?
  - **Product fit**: How credibly can this product own this claim today?
- Multiply scores to produce a Gap Priority Score
- Rank all gaps by score

**Step 5: Draft Positioning Candidates**
- For the top 2–3 gaps, write a positioning statement using this template:
  > "For [target audience] who struggle with [specific unmet need], [product name] is the [category] that [unique claim] — unlike [competitor type] that [limitation]."
- Test each statement against the matrix to confirm no competitor currently owns it strongly

**Step 6: Identify Risks and Validation Steps**
- Flag any gap that competitors may be moving toward based on recent product launches or messaging shifts
- Recommend 1–2 quick validation methods (e.g., run a message test, interview 5 churned customers, A/B test landing page headlines)

## Output Format

Produce a structured report with the following sections:

1. **Competitor Positioning Summary** — A markdown table with competitors as rows and positioning attributes as columns, with ownership strength marked as Strong / Weak / None

2. **Identified Gaps** — Bulleted list of unoccupied or weakly contested positioning spaces, each with a one-sentence rationale citing both competitor absence and customer need evidence

3. **Gap Priority Ranking** — Table showing each gap with Customer Urgency, Competitive Whiteness, Product Fit scores, and final Gap Priority Score, sorted highest to lowest

4. **Positioning Candidates** — 2–3 fully written positioning statements using the template above, one per top-ranked gap

5. **Risk Flags & Validation Steps** — 2–4 bullet points covering competitive movement risks and specific next actions to validate the top positioning choice

Total length: 400–800 words. Use tables and bullets throughout. No paragraphs of prose except inside positioning statements.
