# Marketing Channel Sunset Criteria Builder

This skill creates a concrete, metrics-driven decision framework that defines exactly when a marketing channel should be deprioritized or shut down, removing guesswork and politics from the decision.

## Framework Building Process

### Step 1: Identify the Channel and Its Goals
Ask or confirm:
- Which channel is under evaluation (e.g., paid social, email nurture, podcast sponsorship, SEO, affiliate)
- What was the original business objective (awareness, lead gen, pipeline, revenue)
- How long the channel has been active and what investment has been made

### Step 2: Define Primary Performance Metrics
Select 2–3 KPIs appropriate to the channel type:

**Paid channels:** CAC, ROAS, CPL, conversion rate, MQL-to-SQL ratio
**Organic channels:** traffic growth rate, keyword ranking velocity, organic conversion rate
**Brand/awareness channels:** reach, CPM trends, brand lift, assisted conversions
**Retention channels:** open/click rates, reactivation rate, unsubscribe rate, revenue per send

### Step 3: Set Threshold Values Using This Logic
For each KPI, define three zones:

| Zone | Label | Action |
|------|-------|--------|
| Above target by ≥10% | Green | Continue, increase investment |
| Within ±10% of target | Yellow | Optimize, watch closely |
| Below target by >20% for 2+ consecutive periods | Red | Trigger sunset review |
| Below target by >40% for 3+ consecutive periods | Critical | Initiate sunset |

### Step 4: Establish Time-Based Rules
Define minimum evaluation windows to avoid premature cuts:
- Paid channels: minimum 60–90 days of spend before sunset review
- SEO/content: minimum 6–9 months before organic sunset review
- Email/nurture: minimum 3 full send cycles
- Experimental channels: predefined trial period set at launch (default: 90 days)

### Step 5: Build the Sunset Checklist
Before final sunset decision, require confirmation on:
- [ ] Has the channel had adequate budget and creative testing (minimum 3 variants)?
- [ ] Have external factors (seasonality, algorithm changes) been ruled out?
- [ ] Has a channel specialist reviewed the underperformance root cause?
- [ ] Has leadership been informed of the performance trend?
- [ ] Is there an alternative channel that will absorb this audience or function?

### Step 6: Define the Sunset Decision Rule
Combine conditions into a clear AND/OR rule:

**Sunset is triggered when:**
> [Primary KPI] misses target by [X]% for [N] consecutive [weeks/months] AND the sunset checklist is complete AND no remediation plan has improved performance within [30-day] intervention window.

### Step 7: Document the Wind-Down Protocol
Specify:
- Who approves the sunset decision (role, not name)
- How budget is reallocated
- What data and learnings are archived
- Whether the channel is paused (reversible) or fully decommissioned
