# Social Proof Sequencing Plan

This skill creates a stage-by-stage deployment plan for testimonials, case studies, reviews, and trust signals, mapping each asset type to the right funnel moment to maximize conversion lift.

## Social Proof Sequencing Workflow

### Step 1: Gather Inputs
Ask for or identify the following before building the plan:
- Product/service type and average deal complexity (transactional vs. considered purchase)
- Funnel stages in use (e.g., awareness, consideration, trial, purchase, retention)
- Available social proof assets (testimonials, case studies, star ratings, logos, video reviews, data stats, media mentions, user counts)
- Primary conversion bottlenecks or drop-off points if known
- Target audience segments (e.g., SMB vs. enterprise, B2B vs. B2C)

### Step 2: Classify Social Proof by Type and Weight
Assign each asset to one of four categories:
- **Volume proof** — aggregate ratings, user counts, review counts (builds credibility quickly)
- **Identity proof** — logos, names, photos, titles (builds relatability and trust)
- **Outcome proof** — case studies, ROI stats, before/after results (drives decision-making)
- **Recency proof** — recent reviews, "just joined" notifications, timestamps (reduces hesitation)

### Step 3: Map Proof Types to Funnel Stages
Apply the following sequencing logic:

| Funnel Stage | Goal | Proof Type to Deploy | Placement Examples |
|---|---|---|---|
| **Awareness** | Establish credibility fast | Volume proof + recognizable logos | Homepage hero, ad creative, social posts |
| **Consideration** | Build relevance and trust | Identity proof + short testimonials | Product/feature pages, email nurture, comparison pages |
| **Evaluation** | Overcome objections | Outcome proof (case studies, ROI data) | Pricing page, demo request page, proposal stage |
| **Decision/Purchase** | Eliminate last-mile doubt | Recency proof + specificity (named customer quotes) | Checkout page, cart abandonment email, final CTA area |
| **Onboarding/Retention** | Reinforce the decision | Peer success stories, usage milestones | Welcome emails, in-app prompts, onboarding flows |

### Step 4: Sequence the Deployment Plan
Build a phased rollout:
1. **Phase 1 (Immediate):** Place highest-impact proof at the biggest drop-off point first
2. **Phase 2 (30 days):** Add proof to top-of-funnel to increase qualified traffic trust
3. **Phase 3 (60–90 days):** Develop missing asset types (e.g., create case studies if only ratings exist)
4. **Ongoing:** Refresh recency proof quarterly; rotate testimonials to match audience segments

### Step 5: Flag Gaps and Recommendations
Identify:
- Funnel stages with no social proof coverage
- Asset types that are over-represented (e.g., all testimonials, no outcome data)
- Mismatches (e.g., generic quotes on a high-stakes pricing page)
- Quick wins (e.g., adding a customer count stat to the homepage takes one hour)

## Output Format

Produce a structured plan with these sections:

1. **Social Proof Inventory Summary** — bullet list of available assets categorized by type
2. **Funnel Stage Map** — table showing each funnel stage, current proof coverage, recommended proof type, and specific placement location
