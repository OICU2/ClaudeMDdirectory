# Brand Messaging

This skill builds structured brand messaging frameworks and copy that align brand identity with audience needs, and Claude will use it whenever someone needs to define, refine, or express what their brand stands for.

## Messaging Development Workflow

1. **Extract brand inputs** — Gather or ask for: company name, product/service, target audience, key competitors, and 3 words the brand should embody.

2. **Define the core value proposition** — Write a single sentence: "[Brand] helps [audience] achieve [outcome] by [unique method]." Make it specific, not generic.

3. **Identify audience pain points** — List 3–5 specific frustrations or unmet needs the audience experiences before finding this brand.

4. **Map brand differentiators** — For each pain point, state what this brand does differently from competitors. Avoid vague claims like "best quality" — use concrete proof points.

5. **Establish brand voice** — Define tone using 3 attributes (e.g., "direct, warm, expert") and provide one DO and one DON'T example for each attribute.

6. **Write the messaging pillars** — Create 3 core themes the brand consistently communicates. Each pillar needs: a headline, a one-sentence explanation, and one supporting proof point.

7. **Draft key copy assets** — Produce the following from the framework:
   - Tagline (under 8 words)
   - Elevator pitch (2–3 sentences)
   - Hero headline for a homepage or landing page
   - One-paragraph brand story (past → problem → solution → future)

8. **Check for consistency** — Review all copy against the brand voice attributes. Flag any phrases that contradict the tone or positioning.

## Output Format

Deliver a structured Brand Messaging Document with clearly labeled sections:

```
## Brand Messaging Framework: [Brand Name]

### Core Value Proposition
[One sentence]

### Target Audience
[2–3 sentence profile]

### Pain Points
- [Pain 1]
- [Pain 2]
- [Pain 3]

### Differentiators
- [Competitor does X] → [Brand does Y instead]

### Brand Voice
| Attribute | DO | DON'T |
|-----------|-----|-------|
| [Attribute 1] | ... | ... |

### Messaging Pillars
**Pillar 1: [Theme]**
Headline: ...
Explanation: ...
Proof point: ...

### Copy Assets
**Tagline:** ...
**Elevator Pitch:** ...
**Hero Headline:** ...
**Brand Story:** ...
```

Length: comprehensive but tight — no filler sentences. Every line should be usable copy or a direct input to copy.
