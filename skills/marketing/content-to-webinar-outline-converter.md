# Content to Webinar Outline Converter

Converts long-form written content into a structured, presenter-ready webinar outline with slide sections, talking points, and audience engagement cues.

## Conversion Workflow

1. **Analyze the source content** — Identify the core thesis, main sections, key arguments, supporting data, and actionable takeaways. Note the estimated reading time to gauge webinar length (1 min read ≈ 2–3 min spoken).

2. **Define the webinar structure** — Map the content to a standard webinar arc:
   - **Opening (5–10%)** — Hook, presenter intro, agenda overview, learning outcomes
   - **Body sections (70–80%)** — One slide section per major content section
   - **Engagement breaks (10–15%)** — Polls, Q&A prompts, or reflection moments placed every 10–12 minutes
   - **Close (5–10%)** — Summary, CTA, Q&A open floor, next steps

3. **Build slide section breakdowns** — For each body section, produce:
   - A slide title (6 words or fewer)
   - 3–5 bullet-point talking points the presenter speaks to (not reads)
   - One "slide note" with a stat, story cue, or example to anchor the point

4. **Insert engagement prompts** — Place at least 3 audience interaction moments:
   - Poll question (multiple choice, 3–4 options)
   - Rhetorical or chat-box question
   - Live exercise or quick reflection prompt
   Label each with its type and suggested timing.

5. **Write presenter transitions** — Add a one-sentence bridge between each major section so the flow feels conversational, not segmented.

6. **Generate the CTA and close** — Extract the content's main value proposition and write a closing segment with: a 2-sentence summary, one clear CTA (download, sign up, book a call, etc.), and a teaser for follow-up content if applicable.

## Output Format

Produce a single structured document with the following sections in order:

```
WEBINAR TITLE: [Punchy title derived from source content]
SUBTITLE: [Optional one-line descriptor]
ESTIMATED DURATION: [X minutes]
TARGET AUDIENCE: [Inferred from content]
LEARNING OUTCOMES: [3 bullet points]

---

SEGMENT 1: OPENING ([X] min)
- Hook: [Opening line or question]
- Presenter intro cue: [1 sentence]
- Agenda overview: [3–5 bullet items]

---

SEGMENT 2: [SECTION TITLE] ([X] min)
Slide Title: [≤6 words]
Talking Points:
  • [Point 1]
  • [Point 2]
  • [Point 3]
Slide Note: [Stat, story, or example]
Transition: [One-sentence bridge to next segment]

[Repeat for each body section]

---

ENGAGEMENT PROMPT [#]: [TYPE] — Timing: after Segment [X]
[Prompt text and options if applicable]

---

SEGMENT [FINAL]: CLOSE ([X] min)
Summary: [2 sentences]
CTA: [Specific action]
Follow-up tease: [Optional]
Q&A open: [Suggested duration]
```

Use plain text formatting. Do not use placeholder filler — every field must contain real, derived content from the source material. Flag any sections in the source content that lack enough depth for a full slide with a `[NEEDS EXPANSION]` note.
