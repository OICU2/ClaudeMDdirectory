# Content Distribution Plan

This skill creates a structured, channel-by-channel distribution plan for a content asset, specifying format adaptations, timing, and sequencing to maximize reach and engagement across platforms.

## Distribution Planning Workflow

1. **Identify the core asset** — Determine the primary content piece (blog post, video, podcast, report, etc.), its topic, target audience, and main message.

2. **Audit available channels** — Ask the user which channels they actively use or accept a standard set: email newsletter, LinkedIn, Twitter/X, Instagram, Facebook, YouTube, TikTok, Pinterest, Reddit, podcast, and owned blog/website.

3. **Map format adaptations per channel** — For each active channel, define the specific format transformation required:
   - Long-form blog → LinkedIn article summary, Twitter thread, email digest snippet
   - Video → YouTube full cut, TikTok/Reels short clip, LinkedIn native upload, GIF teaser
   - Podcast → audiogram for Instagram, transcript-based blog, quote cards for Twitter
   - Report/whitepaper → infographic for Pinterest, stat carousel for Instagram, LinkedIn document post

4. **Assign sequencing and timing** — Build a day-by-day posting schedule starting from publish date (Day 0). Use this sequencing logic:
   - Day 0: Publish primary asset on owned channel (blog, YouTube, podcast feed)
   - Day 1: Email newsletter announcement
   - Day 1–2: LinkedIn and Twitter/X primary posts
   - Day 3–5: Instagram and Facebook adapted formats
   - Day 7: Reddit or niche community share (if relevant)
   - Day 14: Reshare or evergreen repost on highest-performing channel
   - Day 30: Recycle as supporting content in a new piece or email sequence

5. **Specify engagement actions** — For each channel post, note required engagement follow-ups: respond to comments within 2 hours, pin top comment, engage in thread replies, tag collaborators or sources.

6. **Flag resource requirements** — List any assets that need to be created per channel (e.g., "need 1080x1080 graphic for Instagram," "need 60-second clip edit for TikTok").

## Output Format

Produce a structured distribution plan with the following sections:

**Content Asset Summary**
- Asset title, type, URL (if available), target audience, core message (1 sentence)

**Channel Distribution Table**
A markdown table with columns: Channel | Format Adaptation | Post Copy Direction | Publish Date/Day | Asset Required

**Day-by-Day Schedule**
A numbered list from Day 0 to Day 30 showing exactly what posts go live, on which channel, and in what format.

**Repurposing Checklist**
Bullet list of every derivative asset to create, with format specs (dimensions, length, file type).

**Engagement Actions**
Short bullet list of post-publish actions per channel (comment responses, reshares, community engagement).

Total output should be comprehensive but scannable — use headers, tables, and bullets throughout. Avoid prose paragraphs except in the asset summary.
