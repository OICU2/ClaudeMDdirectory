# Social Media Crisis Response Playbook

This skill generates a prioritized, time-sequenced response protocol when a brand faces negative viral moments, public complaints, or social media controversies.

## Crisis Response Workflow

### Step 1: Intake — Understand the Situation
Ask for (or extract from context):
- Platform(s) affected (Twitter/X, TikTok, Instagram, Facebook, LinkedIn, Reddit)
- Nature of the crisis: customer complaint, employee misconduct, product failure, offensive content, misinformation, external association
- Current reach/scale: number of posts, estimated impressions, any verified accounts or media involved
- Whether the brand has already responded publicly
- Any factual context the brand knows internally (e.g., what actually happened)

### Step 2: Crisis Classification
Classify severity before drafting response:

- **Level 1 — Contained:** Single viral post, no media pickup, no celebrity/influencer amplification. Response window: 2–4 hours.
- **Level 2 — Escalating:** Multiple posts, some influencer or media interest, trending hashtag forming. Response window: 1–2 hours.
- **Level 3 — Full Crisis:** Trending nationally/globally, media outlets covering, boycott calls, internal leaks. Response window: 30–60 minutes.

### Step 3: Draft the Response Protocol

Produce a sequenced action plan using these phases:

**Phase 1 — Pause & Assess (First 15–30 minutes)**
- Halt all scheduled social posts immediately
- Screenshot and document all negative content
- Identify the original source post and top amplifiers
- Do NOT post an emotional or defensive response
- Assign one internal spokesperson for all external comms

**Phase 2 — Internal Alignment (30–60 minutes)**
- Brief legal, PR, and executive stakeholders on the situation
- Agree on: what is factually true, what is disputed, what the brand is responsible for
- Decide on response posture: Acknowledge / Apologize / Clarify / Dispute (choose one primary stance)
- Determine if a public statement, comment reply, or direct outreach is appropriate

**Phase 3 — Public Response Drafting**
- Write a platform-appropriate statement using this structure:
  1. Acknowledge the concern without minimizing it
  2. State what the brand knows right now (be honest about gaps)
  3. Describe the concrete action being taken
  4. Provide a timeline for follow-up if resolution isn't immediate
- Avoid: legalese, passive voice, blame-shifting, "we're sorry you feel that way" language
- Match tone to platform (LinkedIn = formal, TikTok/Instagram = human and direct)

**Phase 4 — Community Management Actions**
- Identify top 3–5 comments requiring individual replies; draft those replies
- Decide whether to pin a response or update bio link to a statement page
- Flag any comments containing threats or misinformation for moderation review
- Do NOT delete legitimate complaints — this escalates backlash

**Phase 5 — Monitoring & Follow-Through**
- Set up keyword alerts for brand name + crisis terms
- Schedule a 24-hour follow-up post if the crisis involved a real failure or harm
- Conduct internal debrief within 48 hours: what happened, why, and what policy/process changes prevent recurrence

## Output Format

Produce a structured document with the following sections:

1. **Crisis Summary** — 2–3 sentences describing the situation and severity level
2. **Immediate Action Checklist** — Bulleted list of actions for the next 60 minutes, owner-assignable
3. **Drafted Public Statement** — Ready-to-post text, platform-labeled, under 280 characters for Twitter/X or full paragraph for other platforms
4. **Reply Templates** — 3–5
