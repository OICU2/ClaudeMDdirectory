# Marketing Message Consistency Audit

This skill systematically compares messaging across all active marketing channels against a master brand guide to identify contradictions, outdated claims, and tone mismatches, then produces a prioritized remediation report.

## Audit Workflow

### 1. Gather Inputs
Request the following from the user before proceeding:
- Master brand guide (or key excerpts: tone of voice, value props, approved claims, restricted language)
- Channel content samples — collect from as many as apply: website homepage/about/product pages, email campaigns (last 3–6 months), social media bios and recent posts, paid ad copy, sales decks, blog/content hub, press releases
- Any known flagged issues the user has already noticed

If the brand guide is unavailable, ask the user to describe the intended brand voice, core value propositions, and any hard rules (e.g., claims requiring disclaimers, deprecated product names).

### 2. Build a Consistency Baseline
Extract and document from the brand guide:
- **Tone descriptors** (e.g., authoritative but approachable, never casual, always empathetic)
- **Core value propositions** (exact approved phrasing where available)
- **Factual claims** (stats, certifications, product capabilities with version/date)
- **Restricted or deprecated terms** (old product names, discontinued features, banned superlatives)
- **Audience framing** (who the brand speaks to and how it addresses them)

### 3. Audit Each Channel
For every piece of content provided, check against each baseline category:

- **Contradiction check**: Does this channel make a claim that conflicts with another channel or the brand guide? Flag exact quotes from both sources.
- **Outdated claim check**: Does content reference stats, features, pricing tiers, or product names that have since changed? Note the channel, quote, and what has changed.
- **Tone mismatch check**: Does the writing style, formality level, or emotional register deviate from the brand guide's tone descriptors? Cite specific passages.
- **Value prop drift check**: Are core differentiators framed inconsistently or omitted entirely in some channels?
- **Audience framing check**: Does the channel address a different buyer persona or use language misaligned with the intended audience?

### 4. Classify Each Issue
Assign every finding one of three priority levels:
- 🔴 **Critical** — Legal risk, factually wrong claims, direct contradictions between channels, or severe off-brand tone
- 🟡 **Moderate** — Inconsistent framing, outdated but not incorrect claims, tone drift, missing key value props
- 🟢 **Minor** — Stylistic inconsistencies, word choice variations, opportunities to reinforce brand language

---

## Output Format

Produce a structured audit report with the following sections:

### Executive Summary
2–4 sentences covering: total issues found, breakdown by severity, highest-risk channels, and the single most urgent fix.

### Findings by Channel
For each channel audited, create a subsection:

**Channel: [Name]**
| Issue | Severity | Current Copy (exact quote) | Conflicts With | Recommended Fix |
|-------|----------|---------------------------|----------------|-----------------|
| [type] | 🔴/🟡/🟢 | "[quote]" | [brand guide section or other channel] | [specific rewrite or action] |

### Cross-Channel Contradictions
A dedicated table listing any cases where two or more channels make conflicting statements about the same topic, with quotes from each.

### Prioritized Remediation List
Numbered action list ordered by severity, then effort. Each item includes:
- What to fix
- Which channel(s) affected
- Owner suggestion (e.g., web team, social manager, legal review)
- Estimated complexity: Quick fix / Requires approval / Full rewrite
