# SEO Content Brief Generation

This skill produces a structured, research-backed content brief for a target keyword that a writer can use to create content optimized for search rankings and user intent.

## Content Brief Workflow

**1. Keyword & Intent Analysis**
- Identify the primary keyword and 3–5 semantically related secondary keywords (LSI terms)
- Classify search intent: informational, navigational, commercial, or transactional
- Define the target audience persona (role, pain point, knowledge level)
- Note dominant SERP features for this keyword (featured snippet, People Also Ask, local pack, etc.)

**2. Competitor Coverage Audit**
- Identify 3–5 typical top-ranking page types for this keyword (listicle, how-to guide, comparison, landing page, etc.)
- List topics and subtopics that appear consistently across top results (must-cover)
- Identify coverage gaps — angles, questions, or subtopics that top results miss or handle poorly
- Note one unique angle or differentiator the content should own

**3. Content Structure & Headings**
- Recommend a content type (e.g., ultimate guide, step-by-step tutorial, comparison article)
- Propose a full H1–H4 heading hierarchy with specific, descriptive heading text (not placeholders)
- Flag which sections address People Also Ask questions or featured snippet opportunities
- Specify recommended word count range based on content type and intent

**4. On-Page SEO Requirements**
- Suggest a meta title (under 60 characters) and meta description (under 155 characters)
- List 3–5 terms that must appear in the body for topical completeness
- Recommend image alt text themes and any schema markup type (HowTo, FAQ, Article, etc.)
- Specify where the primary keyword should appear: H1, first 100 words, at least one H2

**5. Internal Linking Suggestions**
- Suggest 3–5 internal link opportunities by describing the type of existing content to link from (e.g., "pillar page on [broader topic]," "comparison article covering [related tool]")
- Suggest 1–2 pages this new content should link out to internally
- Identify where in the new content each internal link anchor should appear

**6. Call-to-Action & Conversion Goal**
- Define the primary CTA aligned with search intent (e.g., download checklist, start free trial, contact sales, read next article)
- Specify placement: above the fold, mid-content, end of article

## Output Format

Produce the brief as a clearly sectioned document with the following structure:

```
# Content Brief: [Primary Keyword]

## Overview
- Primary Keyword:
- Secondary Keywords:
- Search Intent:
- Target Audience:
- Recommended Content Type:
- Target Word Count:
- SERP Features to Target:

## Competitor Coverage Gaps
[3–5 bullet gaps with a one-line explanation each]

## Unique Angle
[1–2 sentences defining what makes this piece different]

## Recommended Heading Structure
H1: [specific title]
  H2: [section]
    H3: [subsection]
  H2: [section]
  ...

## On-Page SEO
- Meta Title:
- Meta Description:
- Must-Include Terms:
- Schema Type:
- Keyword Placement Checklist:

## Internal Linking Plan
- Link From: [content type] → Anchor: "[suggested anchor text]" → Place in: [section name]
[repeat for each suggestion]
- Link Out To: [content type] from [section name]

## CTA & Conversion Goal
- Primary CTA:
- Placement:
```

The brief must be usable by a writer without additional research — every section should contain specific, filled-in recommendations, not instructions to the writer to go find information themselves.
